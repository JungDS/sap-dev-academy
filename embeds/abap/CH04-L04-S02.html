PARAMETERS pa_grade TYPE c LENGTH 1.

CASE pa_grade.
  WHEN 'A'.
    WRITE: / '우수'.
  WHEN 'B'.
    WRITE: / '양호'.
  WHEN 'C'.
    WRITE: / '보통'.
  WHEN OTHERS.
    WRITE: / '등급 없음'.
ENDCASE.
  • 마름모 대신 한 값을 여러 WHEN과 = 비교
  • 처음 맞는 가지만 실행하고 ENDCASE로
  • 아무 데도 안 맞으면 WHEN OTHERS
