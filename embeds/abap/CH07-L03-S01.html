같은 모양, 두 운명 (메모리 vs 디스크)

똑같은 dan·mul·result 한 줄이라도 어디에 두느냐로 수명이 갈립니다. 둘 다 옳은 도구 — 쓰임이 다를 뿐입니다.

🧠 메모리 — 휘발 (Internal Table)
" 메모리(내부 테이블)
DATA gt_gugu TYPE TABLE OF ty_line.

APPEND gs_line TO gt_gugu.   " 81행 쌓기
" 프로그램이 끝나면 → gt_gugu 통째로 사라짐
  • 실행이 끝나면 흔적도 없이 소멸
  • 다시 실행하면 처음부터 새로 채워야
  • 다른 사용자·세션은 볼 수 없음(내 실행 안에서만)
수명: 실행하는 동안만
💾 디스크 — 영속 (Transparent Table)
" 디스크(Transparent Table) — SE11
TABLE ZGUGUDAN
  DAN     INT4
  MUL     INT4
  RESULT  INT4
" 활성화 → 디스크에 영구 생성
  • 활성화 후 디스크에 영구 보존
  • 프로그램이 끝나도, 껐다 켜도 그대로
  • 다른 사용자·세션도 같은 데이터를 조회
수명: 영구
