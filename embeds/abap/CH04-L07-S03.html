3단계 — 범위만큼만 반복 (예: 3~5단)

1~9를 다 돌며 거르지 말고, p_from부터 필요한 횟수만큼만 돌고 회차 끝에 단을 1 올립니다. 정답을 보기 전에 빈칸을 직접 채워 보세요.

ABAP
PARAMETERS: p_from TYPE i, p_to TYPE i.
DATA: gv_dan TYPE i, gv_mul TYPE i, gv_cnt TYPE i.

gv_cnt = p_to - p_from + 1.   " 출력할 단 개수
gv_dan = .            " 시작 단부터

DO  TIMES.
  DO 9 TIMES.
    gv_mul = sy-index.
    WRITE: / gv_dan, 'x', gv_mul, '=', gv_dan * gv_mul.
  ENDDO.
  gv_dan = gv_dan + .        " 다음 단으로
ENDDO.
