WHILE — 조건이 참인 동안 (gv_n ≤ 5)

왼쪽은 반복 흐름, 오른쪽은 그 ABAP 코드입니다. 조건을 먼저 검사해 참일 때만 본문을 실행하고, 거짓이 되면 멈춥니다.

조건 검사 → 본문 → 되돌아가기

본문 안의 gv_n + 1이 언젠가 조건을 거짓으로 만들어야 멈춥니다. 없으면 무한루프.

DATA gv_n TYPE i VALUE 1.

WHILE gv_n <= 5.
  WRITE: / gv_n.
  gv_n = gv_n + 1.     " 없으면 무한루프!
ENDWHILE.
  • 조건부터 검사 — 참일 때만 본문 실행
  • gv_n + 1이 조건을 언젠가 거짓으로 만듦
  • 증가를 빠뜨리면 영원히 반복(무한루프)
