ZC_Booking 해부 — Projection + @UI(CDS)

밑줄 부분을 클릭하세요. @UI 주석이 Fiori 화면(목록·상세)을 코드 없이 선언합니다.

ZC_Booking (CDS View · Projection)
define root view entity ZC_Booking
  provider contract transactional_query
  as projection on ZI_Booking
{
  @UI.facet: [{ type: #IDENTIFICATION_REFERENCE, position: 10 }]
  key booking_id,

  @UI: { lineItem: [{ position: 10 }],
      identification: [{ position: 10 }] }
  @EndUserText.label: '공연'
      concert_id,
      customer, seats, status, _Concert, _Perf
}
밑줄 친 구문을 클릭하면 설명이 나옵니다.
💡 @UI.lineItem=목록(List Report) 컬럼 · @UI.identification/facet=상세(Object Page) 폼. 주석이 많으면 Metadata Extension(CH22-L05)으로 분리합니다.
