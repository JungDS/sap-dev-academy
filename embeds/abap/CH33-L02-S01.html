AMDP 메서드 해부 — 부분을 클릭해 보기

밑줄 친 부분을 클릭하면 그 구문이 무슨 역할인지 설명이 나옵니다. AMDP는 ABAP 메서드 안에 SQLScript를 담아 DB에서 실행합니다.

ZCL_BOOKING_AMDP (ABAP · AMDP)
CLASS zcl_booking_amdp DEFINITION.
  PUBLIC SECTION.
    INTERFACES if_amdp_marker_hdb.
    METHODS get_stats
      IMPORTING VALUE(iv_concert) TYPE zconcert-concert_id
      EXPORTING VALUE(et_stats)   TYPE ztt_stats.
ENDCLASS.

CLASS zcl_booking_amdp IMPLEMENTATION.
  METHOD get_stats BY DATABASE PROCEDURE FOR HDB LANGUAGE SQLSCRIPT
                   USING zbooking OPTIONS READ-ONLY.
    et_stats = SELECT concert_id, SUM( seats ) ... GROUP BY concert_id
               FROM zbooking
               WHERE concert_id = :iv_concert;
  ENDMETHOD.
ENDCLASS.
밑줄 친 구문을 클릭하면 여기에 설명이 나옵니다.
💡 본문은 SQLScript라 ABAP 문을 못 씁니다. 접근 테이블은 USING에 선언해야 하고, 읽기만 하면 READ-ONLY로 더 빠르고 안전하게.
