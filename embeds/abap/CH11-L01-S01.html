WRITE 리스트 vs SALV 표 (같은 데이터, 다른 화면)

같은 예매 6행을 두 방식으로 봅니다. WRITE는 값은 보이지만 사용자가 정렬·필터·합계를 할 수 없습니다. SALV는 같은 Internal Table을 조작 가능한 표로 만들어 줍니다.

정렬필터Σ 합계엑셀인쇄
WRITE 리스트
✕정렬 — 직접 코딩
✕필터 — 직접 코딩
✕합계 — 직접 계산·출력
✕엑셀 내보내기 — 별도 구현
SALV 표
✓정렬 — 표준 기능
✓필터 — 표준 기능
✓합계 — 숫자 컬럼 표준
✓엑셀 — 표준 툴바 (읽기 전용)
위 버튼으로 정렬·합계를 눌러 두 화면의 차이를 확인하세요.
경계 — SALV는 데이터 조회 도구가 아닙니다. 조회는 SELECT(CH08), 담는 곳은 Internal Table(CH06), 보여 주는 건 SALV. CH11 SALV 1차는 읽기 전용 — 셀 편집은 CH31에서 배웁니다.
