콘서트앱 — classic ↔ modern 리팩터 (인라인 DATA · += · VALUE · Table Expr)

CH10-L07 잔여석 합산 등 콘서트앱 코드를 모던으로 다듬습니다. 줄에 마우스를 올려 대응을 확인하세요. (SELECT 자체의 모던화는 @가 필요한 CH19 — 여기선 SQL 밖 ABAP 코드만.)

🔁 classic ↔ modern — 줄 위에 마우스를 올려 보세요

classic

DATA: ls_book TYPE zbooking, lv_sum TYPE i.
LOOP AT lt_book INTO ls_book.
  lv_sum = lv_sum + ls_book-seats.
ENDLOOP.
CLEAR ls_new.
ls_new-concert_id = 'C001'.
ls_new-customer   = '정훈영'.
ls_new-seats      = 2.
READ TABLE lt_book INTO ls_book
  WITH KEY booking_id = lv_id.

modern

LOOP AT lt_book INTO DATA(ls).
  lv_total += ls-seats.
ENDLOOP.
DATA(ls_new) = VALUE zbooking(
  concert_id = 'C001' customer = '정훈영'
  seats = 2 status = 'N' ).
DATA(ls_hit) = lt_book[ booking_id = lv_id ].
줄 위에 마우스를 올리면 바뀐 이유가 여기 표시됩니다.
