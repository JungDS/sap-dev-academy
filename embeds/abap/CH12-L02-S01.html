SELECT-OPTIONS 화면 변환기 (입력 → Range Table → LOOP)

화면의 From/To 칸과 다중 선택이 어떻게 s_conc Range Table 행으로 바뀌는지 봅니다. From만 넣으면 EQ, To까지 넣으면 BT입니다.

선택화면 (모형)

s_conc ~ FOR zbooking-concert_id
다중 선택 팝업 — 여러 값·범위·제외를 행으로 추가
↓ Range Table s_conc 로 변환
SIGNOPTIONLOWHIGH출처
ABAP LOOP AT s_conc INTO ls. WRITE: / sign option low high.classic

  


  
화면 = 프로그램. 입력칸 하나가 SIGN/OPTION/LOW/HIGH 행이 됩니다. From만 = EQ(LOW만), From~To = BT(LOW·HIGH). 비우면 행이 없어 LOOP이 0회 — 조건을 걸지 않은 것과 같습니다.
