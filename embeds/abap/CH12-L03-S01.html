WHERE … IN 필터 엔진 (so_conc · so_stat → sy-subrc · sy-dbcnt)

Range Table을 WHERE field IN so_xxx로 조회에 꽂으면 zbooking 목록이 줄어듭니다. 두 조건 사이는 AND입니다. 빈 범위는 true라 비우면 전체가 나옵니다. 실행 후 sy-subrc·sy-dbcnt를 확인하세요.

시나리오:
so_conc FOR zbooking-concert_id
so_stat FOR zbooking-status (N 예약 · C 취소)

빈 range = true. 비운 조건은 제한하지 않아 전체가 통과합니다(실수로 비우면 전체 조회 위험 — 입력 필수 검증은 CH15). 두 줄 사이 AND라 둘 다 만족해야 합니다. 결과가 없으면 sy-subrc = 4, 있으면 0 · sy-dbcnt = 담긴 행 수. @·콤마 없는 classic 표기 — modern은 CH19.
