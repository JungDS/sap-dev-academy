PDF 생성·다운로드 해부 — 부분을 클릭

Adobe Forms로 PDF 바이트를 받아 파일로 저장하는 코드입니다. 밑줄 부분을 클릭해 역할을 확인하세요.

예매 확인서 PDF 저장 (ABAP)
CALL FUNCTION 'FP_JOB_OPEN' CHANGING ie_outputparams = ls_out.
CALL FUNCTION lv_fm EXPORTING is_booking = ls_booking
  IMPORTING /1bcdwb/formoutput = ls_formout.
CALL FUNCTION 'FP_JOB_CLOSE'.

DATA(lv_pdf) = ls_formout-pdf.   " XSTRING(PDF 바이트)
DATA lt_bin TYPE solix_tab.
lt_bin = cl_bcs_convert=>xstring_to_solix( lv_pdf ).
cl_gui_frontend_services=>gui_download(
  EXPORTING filename = 'ticket.pdf' filetype = 'BIN'
            bin_filesize = xstrlen( lv_pdf )
  CHANGING  data_tab = lt_bin ).
밑줄 친 구문을 클릭하면 여기에 설명이 나옵니다.
💡 PDF는 텍스트가 아니라 바이너리(XSTRING) — 저장은 반드시 BIN 모드. 출력은 FP_JOB_OPEN~FP_JOB_CLOSE 사이에서. 이메일 첨부는 BCS, 보관은 ArchiveLink/DMS.
