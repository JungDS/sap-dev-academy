READ TABLE vs Table Expression (예외 · line_exists · line_index)

검색 키를 골라 네 가지 접근을 비교하세요. B001/B002는 존재, B999는 없습니다. READ TABLE은 sy-subrc만 세팅하지만, lt[ ]는 없는 행을 직접 읽으면 예외가 납니다.

검색 키 booking_id

네 가지 접근 결과

짧지만 안전하게. Table expression lt[ ]은 한 행을 값처럼 읽지만, 성공 여부는 sy-subrc가 아니라 line_exists( )로 확인합니다. 없는 행을 직접 읽으면 CX_SY_ITAB_LINE_NOT_FOUND 예외 — line_exists로 먼저 확인하세요.
