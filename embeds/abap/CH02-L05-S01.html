매직 넘버 vs 상수 (같은 코드, 두 버전)

코드에 박힌 정체불명의 숫자(매직 넘버)를 CONSTANTS로 이름 붙이면 어떻게 달라지는지 나란히 비교해 보세요.

✗ 매직 넘버
DATA lv_amt TYPE i VALUE 1500.

IF lv_amt > 1000.            " 이 1000이 한도? 수량?
  WRITE: / '한도 초과 (최대 1000)'.
ENDIF.
  • 정체불명의 1000이 여러 곳에 흩어짐
  • 한도가 2000으로 바뀌면 빠짐없이 찾아 고쳐야(하나라도 놓치면 버그)
한도 변경 시 고칠 곳: 2곳
✓ CONSTANTS
CONSTANTS gc_max_amount TYPE i VALUE 1000.

IF lv_amt > gc_max_amount.   " 이름으로 읽힘
  WRITE: / '한도 초과'.
ENDIF.
  • 값에 이름을 붙여 뜻이 분명
  • 한도 변경 = 선언 한 줄(VALUE)만 — 쓰는 곳은 그대로
한도 변경 시 고칠 곳: 1곳
