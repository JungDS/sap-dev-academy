CALL FUNCTION 상자 (호출자 기준 계약)

Function Module은 검은 상자가 아니라 계약 상자입니다. EXPORTING·IMPORTING은 반드시 호출자 기준으로 읽습니다 — 내가 내보내고(EXPORTING), 내가 받아옵니다(IMPORTING). 시나리오를 실행해 sy-subrc를 확인하세요.

호출자 (zch10_l03)
CALL FUNCTION 'Z_CH10_ADD_TAX'
주의 2가지 — ① EXPORTING/IMPORTING은 호출자 기준으로 읽기(위 토글로 틀린 읽기 확인). ② CALL FUNCTION 뒤 sy-subrc 분기 잊지 않기. TABLES·일부 obsolete typing은 기존 FM에서 보일 뿐 신규 권장 아님.
