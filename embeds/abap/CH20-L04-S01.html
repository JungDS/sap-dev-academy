클래스명 왼쪽 → => (정적 멤버). 객체 없이 호출.
참조 변수 왼쪽 → -> (인스턴스 멤버). 객체가 필요.
me->는 메서드 안에서 자기 객체의 멤버를 가리킬 때 씁니다(보통 생략 가능, 지역변수·파라미터와 이름이 비슷할 때 의도를 분명히). 정적 메서드 안에는 자기 객체가 없어 me->를 쓸 수 없습니다.
