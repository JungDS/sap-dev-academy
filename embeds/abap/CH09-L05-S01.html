Collective Search Help 탭 시뮬레이터 (F4 여러 검색 방식 묶기)

사용자는 공연 ID를 알 수도, 아티스트만 기억할 수도, 장소만 알 수도 있습니다. 여러 Elementary Search Help를 탭으로 묶어, 어느 탭에서 골라도 같은 입력칸으로 CONCERT_ID가 돌아옵니다.

ZSH_CONCERT_ALL (Collective)
🔍 공연 검색 (Collective Search Help)
입력칸 반환값 →
—
행을 고르면 CONCERT_ID가 반환됩니다.
핵심 — Collective는 "검색 도움말 여러 개를 모은 폴더"입니다. 단, 폴더 안 Elementary들이 같은 반환 parameter로 연결돼야 어느 탭에서 골라도 같은 값이 돌아옵니다. 검색 방식이 하나뿐이면 Collective는 과설계입니다.
