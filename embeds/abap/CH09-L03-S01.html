언어별 이름표 뷰어 (Text Table · SPRAS)

코드(C001)는 마스터 ZCONCERT에, 사람이 읽을 이름은 언어별 Text Table ZCONCERT_T에 둡니다. 로그인 언어를 바꾸면 같은 코드의 F4 이름이 달라집니다.

로그인 언어
마스터 · ZCONCERT
Text Table · ZCONCERT_T
F4 결과 목록 (현재 언어)
핵심 — Text Table의 키 = 마스터 키 + 언어 키(SPRAS). 언어 키를 빠뜨리면 다국어 텍스트를 담을 수 없습니다. 코드는 마스터에, 이름은 Text Table에 — F4 목록에서 코드 옆에 이름이 붙는 이유입니다.
