classic ↔ modern — 같은 로직, 두 표기 (VALUE · +=)

왼쪽 classic 줄에 마우스를 올리면 오른쪽 modern 대응 줄이 함께 빛나고, 아래에 무엇이 어떻게 짧아졌는지가 나타납니다. classic도 현장에 있으니 읽을 줄 알아야 합니다.

🔁 classic ↔ modern — 줄 위에 마우스를 올려 보세요

classic (개선 전)

DATA ls TYPE ty_line.
DATA lt TYPE ty_line_tab.
ls-dan = 2. ls-mul = 1. ls-result = 2.
APPEND ls TO lt.
lv_total = lv_total + ls-result.

modern (권장)

DATA(lt) = VALUE ty_line_tab(
  ( dan = 2 mul = 1 result = 2 ) ).
lv_total += ls-result.
줄 위에 마우스를 올리면 바뀐 이유가 여기 표시됩니다.
