Screen Painter 연결 지도 (화면 0100)

Dynpro는 Layout · Element List · Flow Logic · ABAP Source 네 조각이 이름으로 연결됩니다. 시나리오를 골라 연결을 하나씩 깨뜨려, 어떤 활성화 오류가 나는지 확인하세요.

시나리오 (연결 깨뜨리기)

이름이 다리다. Flow Logic의 MODULE status_0100.(호출)과 ABAP의 MODULE status_0100 OUTPUT.(정의)은 다른 위치에 있고 이름이 같아야 연결됩니다. 화면 요소 이름 ↔ 변수 이름, OK field 지정도 모두 맞아야 활성화됩니다.
