Concert Advanced SQL Lab (CH20 통합 실습)

콘서트 예매 두 테이블(zconcert·zbooking)로 이번 챕터의 고급 SQL 네 가지를 직접 돌려 봐요. 버튼으로 기법을 바꾸고, 조건 제거 실험으로 취소된 예약(status = 'C')이 결과를 어떻게 망가뜨리는지 확인합니다.

업무 규칙이 곧 SQL 조건이다. "취소된 예약은 좌석을 차지하지 않는다"는 규칙 하나가 빠지면 잔여석·예약목록·집합·순번이 전부 어긋나요. 고급 SQL일수록 WHERE status <> 'C' 같은 업무 조건을 정확히 거는 것이 핵심입니다.
