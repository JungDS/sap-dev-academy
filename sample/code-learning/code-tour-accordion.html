코드에서 시선 떼지 않고 의미 보기

점선 밑줄 키워드를 클릭하면, 그 줄 아래로 설명이 펼쳐집니다. 코드를 떠나지 않고 맥락 안에서 개념을 익힙니다.

ZDEMO_SELECT.abap
DATA lt_emp▾ TYPE STANDARD TABLE OF pa0001.

SELECT pernr name1 FROM pa0001
  INTO CORRESPONDING FIELDS OF TABLE▾ lt_emp
  WHERE▾ werks = '1000'.

IF sy-subrc▾ = 0.
  WRITE / |{ lines( lt_emp ) } 건 조회됨|.
ENDIF.
lt_emp — 내부 테이블
같은 구조의 여러 행을 메모리에 담는 그릇입니다. STANDARD TABLE OF pa0001은 "PA0001 한 줄과 동일한 모양의 행"을 여러 개 담겠다는 선언입니다.
INTO CORRESPONDING FIELDS OF TABLE
SELECT 결과를, 이름이 같은 컬럼끼리 자동 매핑해 테이블에 채웁니다. 컬럼 순서가 달라도 이름으로 맞춰 줍니다.
WHERE — 필터 조건
DB에서 가져올 행을 좁힙니다. 여기선 werks = '1000'(사업장 1000)인 행만 가져옵니다. 가능한 한 DB 단에서 걸러야 성능에 좋습니다.
sy-subrc — 결과 코드
직전 문장의 성공/실패를 담는 시스템 필드. 0이면 한 건 이상 조회됨, 4면 결과 없음을 뜻합니다.
