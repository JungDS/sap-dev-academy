WRITE 포맷 플레이그라운드 (폭·정렬·색·강조)

값과 포맷 옵션을 바꾸면, 왼쪽에 WRITE 코드가 만들어지고 오른쪽 SAP 리스트에 결과가 그려집니다. 같은 값이라도 RIGHT-JUSTIFIED·COLOR·ULINE으로 얼마나 달라지는지 보세요.

예제:
⚙️ 포맷 옵션
16
생성된 ABAP · ZHELLO

      

      
🖥️ 리스트 출력

      

    

  


  
💡 폭(len) 안에서 값을 어디로 붙일지가 정렬(LEFT·CENTERED·RIGHT-JUSTIFIED)입니다. 숫자는 보통 RIGHT-JUSTIFIED로 자릿수를 맞추고, 합계·경고는 COLOR로 눈에 띄게 합니다. ULINE은 밑줄선, SKIP은 빈 줄을 넣습니다.
