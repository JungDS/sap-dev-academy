루프 내부 SELECT 제거하기

"무엇이 왜 나빴고, 어떻게 좋아졌는지"를 나란히 보여 주면 개선의 효과가 또렷해집니다.

✗ 개선 전 — N+1 쿼리
LOOP AT lt_ord INTO ls.
  SELECT SINGLE name1
    FROM pa0001 INTO lv
    WHERE pernr = ls-pernr.
ENDLOOP.
  • 행마다 DB 왕복 → 1,000행이면 1,000번
  • 네트워크/파싱 오버헤드 누적
DB 호출 1,000회
✓ 개선 후 — 일괄 조회
SELECT pernr name1
  FROM pa0001
  INTO TABLE lt_name
  FOR ALL ENTRIES IN lt_ord
  WHERE pernr = lt_ord-pernr.
" 이후 READ TABLE 로 매칭
  • 한 번에 가져와 메모리에서 매칭
  • DB 왕복 1회로 축소
DB 호출 1회
