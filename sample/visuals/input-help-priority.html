Input Help · F4

F4를 누르면 — 위에서부터 본다

한 입력칸에 도움말 장치가 여러 개 걸려 있어도, SAP은 위 순서대로 딱 하나를 골라 띄운다. 위가 있으면 아래는 안 본다.

🥇 가장 먼저 본다↓ 없으면 다음으로나중에 배움 = 지금은 이름만
1
코드로 직접 만든 F4 PROCESS ON VALUE-REQUEST CH15·16에서 배움

프로그램이 화면 이벤트에서 목록을 직접 만들어 띄운다. 있으면 무조건 이게 최우선 — 지금은 "이런 게 맨 위에 있다"만.

▼ 없으면
2
Search Help CH09

설계된 검색 화면(Elementary·Collective). 가장 유연한 선언적 도움말 — 필드·구조·Data Element에 붙는다.

▼ 없으면
3
Check Table · Text Table · 고정값 CH09

Foreign Key의 Check Table 목록, 코드 옆 이름(Text Table), Domain의 Fixed Values. DDIC 관계·값에서 자동으로 목록을 만든다.

▼ 없으면
4
타입 기본 도움 자동

날짜 타입이면 달력, 숫자면 계산기처럼 데이터 타입이 주는 기본 도움. 아무것도 안 걸어도 이건 뜬다.

💡 그래서 한 칸에 도움말을 여러 개 겹쳐 걸면 의도와 다른 게 뜰 수 있다. 보통은 하나로 명확히(예: MATCHCODE OBJECT로 Search Help 명시) 하는 게 좋다.
