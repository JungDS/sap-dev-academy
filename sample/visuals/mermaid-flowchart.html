텍스트 문법으로 그리는 다이어그램이라 코드처럼 버전 관리됩니다. 실선=정상 경로, 점선=예외(검증 실패 복귀)로 한 장에 두 경로를 담았습니다.

실행 순서와 검증 실패 복귀 루프

flowchart TD Start([Program Starts]) --> INIT[INITIALIZATION
화면 기본값 세팅] INIT --> Screen[Selection Screen
사용자 입력 대기] Screen -->|입력 발생| ASS{AT SELECTION-SCREEN
입력값 검증} ASS -.->|에러 시 복귀| Screen ASS -->|통과 + F8| SOS[START-OF-SELECTION
