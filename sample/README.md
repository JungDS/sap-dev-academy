# Sample Library — 학습 수단 / 웹페이지 구조 샘플

`C:\ui5\study\sapui5`(구 참고 repo)의 웹페이지에서 **이 프로젝트(SAP Developer Academy)에 적용하면 좋을 요소**들을
선별·재구성한 standalone 샘플 모음입니다.

## 핵심 원칙

- **각 HTML은 완전 self-contained 단일 파일**입니다. CSS·JS를 모두 `<style>`/`<script>`로 내장했습니다.
- 따라서 특정 샘플을 쓰고 싶을 때 **그 파일 하나만 복사**하면 됩니다. 외부 의존성이 없습니다.
- 예외(외부 CDN 1개)는 파일 상단 주석과 아래 표의 **의존성** 열에 명시했습니다.
- 모든 샘플은 현재 프로젝트의 디자인 토큰(`assets/base.css`의 `--brand:#3b5bdb`, `--accent:#0ca678` 등)을
  그대로 따르므로, 본 사이트에 이식해도 톤이 어긋나지 않습니다.

## 사용법

1. `sample/index.html`을 브라우저로 열면 전체 카탈로그를 카드로 볼 수 있습니다.
2. 마음에 드는 샘플의 HTML 파일을 열어 동작을 확인합니다.
3. 그 파일의 `<style>`(컴포넌트 CSS)과 `<script>`(컴포넌트 JS) 블록을 본문 템플릿/뷰어로 옮겨 붙이면 끝입니다.
   - 파일마다 `/* ===== 컴포넌트 CSS ===== */`, `// ===== 컴포넌트 JS =====` 주석으로 "공통 셸"과 "컴포넌트 고유" 코드를 구분했습니다.
   - 재사용할 땐 **컴포넌트 고유** 블록만 가져가면 됩니다(공통 셸은 데모용).
