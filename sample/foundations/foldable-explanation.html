DDIC 가 뭔가요?기본

DDIC(Data Dictionary)는 SAP의 중앙 데이터 정의 사전입니다. 테이블·데이터 타입·도메인을 한 곳에서 정의해 전체 시스템이 공유합니다.

왜 ABAP 변수 타입을 DDIC와 연결하나요?심화

DATA lv_pernr TYPE pa0001-pernr. 처럼 선언하면, 테이블 컬럼의 길이·타입이 바뀌어도 코드가 자동으로 따라갑니다.

즉 단일 진실 공급원(Single Source of Truth)이 DDIC가 되어 유지보수 비용이 줄어듭니다.

도메인(Domain)과 데이터 요소(Data Element)의 차이는?심화

도메인 = 기술적 속성(타입·길이·값 범위). 데이터 요소 = 도메인 + 의미(필드 라벨·문서).

비유하면 도메인은 "그릇의 규격", 데이터 요소는 "그 그릇에 붙인 이름표"입니다.
