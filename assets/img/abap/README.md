# assets/img/abap — ABAP 레슨 이미지(스크린샷 등)

레슨 본문이 `image-figure` 임베드로 참조하는 실제 화면 스크린샷을 둔다.
임베드(`embeds/abap/*.html`)는 `../../assets/img/abap/<파일>`로 참조(= 이 폴더).

## 필요한 파일 (사용자가 직접 추가)
| 파일명 | 내용 | 사용처 |
|---|---|---|
| `CH01-L01-logon.png` | SAP 로그온(로그인) 화면 | CH01-L01 §03, 임베드 `CH01-L01-S01` |
| `CH01-L01-easy-access.png` | SAP Easy Access 첫 화면 | CH01-L01 §04, 임베드 `CH01-L01-S02` |

> 파일을 위 경로에 저장하면 레슨에 자동 표시된다(빌드 불필요 — 임베드가 런타임에 로드).
> 파일이 없으면 임베드가 "이미지를 추가하세요" 플레이스홀더를 보여 준다.
