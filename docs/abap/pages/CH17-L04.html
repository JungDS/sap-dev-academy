Field Catalog 기초

컬럼을 정의한다 — Field Catalog로 제목·순서·속성 제어.

Track 1Chapter 17Lesson 4중급

컬럼을 내 맘대로

(필드 카탈로그)는 "어떤 컬럼을, 어떤 제목·너비·순서·정렬로" 보여줄지 정의하는 표다(타입 LVC_T_FCAT). 그리드의 핵심 설정이다.

자동으로 만들기 — merge

행 구조가 DDIC에 있으면, 함수로 를 자동 생성할 수 있다.

ABAP

DATA lt_fcat TYPE lvc_t_fcat.

CALL FUNCTION 'LVC_FIELDCATALOG_MERGE'
  EXPORTING i_structure_name = 'ZBOOKING'
  CHANGING  ct_fieldcat      = lt_fcat.

손으로 다듬기

자동 생성 후, 특정 컬럼의 제목·너비·숨김 등을 코드로 조정한다.

DATA ls_fcat TYPE lvc_s_fcat.

LOOP AT lt_fcat INTO ls_fcat.
  IF ls_fcat-fieldname = 'SEATS'.
    ls_fcat-coltext = '좌석수'.       " 컬럼 제목
    ls_fcat-outputlen = 8.            " 너비
    MODIFY lt_fcat FROM ls_fcat.
  ENDIF.
ENDLOOP.

흔한 실수와 점검

  • fieldcat 비움 → 컬럼 정의가 없어 표가 안 그려진다. merge 또는 수동 작성.
  • fieldname 대문자 → 보통 DDIC 필드명은 대문자('SEATS').

컬럼을 정의했다. 이제 표 전체의 모양(줄무늬·선택 모드 등)을 정하는 Layout → 다음 레슨.
