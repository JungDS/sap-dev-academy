Output Control 개요

언제 무엇을 누구에게 출력하나 — 출력 결정.

Track 2Chapter 34Lesson 3중급

출력은 자동으로 결정된다

"주문이 저장되면 확인서를 고객에게 이메일·인쇄"처럼, 언제·무엇을·누구에게·어떻게 출력할지는 Output Control이 조건으로 결정한다.

  • 전통(NAST) — 메시지 결정 절차(조건 ·출력 유형).
  • 현대(BRF+ 기반 Output Management, S/4HANA) — 규칙으로 출력 결정.
  • 결정되면 연결된 양식(Smart/Adobe)과 채널(인쇄·이메일·EDI)로 나간다.

흔한 실수와 점검

  • 양식만 만들고 결정 없음 → 자동 출력이 안 된다. Output 조건을 설정.
  • NAST vs BRF+ → 시스템 버전에 맞는 방식 사용(S/4는 BRF+ 권장).
