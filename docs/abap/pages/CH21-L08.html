회차 현황을 ALV로 띄우되, 매진은 빨강·잔여 적으면 노랑으로 강조한다(CH21-L04 셀색).

TYPES: BEGIN OF ty_row,
         concert_id TYPE zconcert-concert_id,
         perf_no    TYPE zbooking-perf_no,
         capacity   TYPE i,
         left       TYPE i,
         cellcolors TYPE lvc_t_scol,
       END OF ty_row.

LOOP AT lt_row ASSIGNING FIELD-SYMBOL(<r>).
  DATA(lo_mgr) = NEW zcl_booking_manager(
    iv_concert = <r>concert_id iv_perf = <r>perf_no ).
  <r>left = lo_mgr->remaining( ).            " CH20-L10
  IF <r>left <= 0.
    <r>cellcolors = VALUE #( ( fname = 'LEFT' color-col = col_negative ) ). " 빨강
  ELSEIF <r>left <= 5.
    <r>cellcolors = VALUE #( ( fname = 'LEFT' color-col = col_total ) ).    " 노랑
  ENDIF.
ENDLOOP.

ls_layout-ctab_fname = 'CELLCOLORS'.

직접 해 볼 과제

  1. 회차별 잔여석을 zcl_booking_manager(CH20-L10)로 계산해 채운다.
  2. 매진(0)·잔여 적음(≤5)에 색이 입혀지는지 확인.
  3. (도전) 더블클릭 시 그 회차 예매 목록을 띄운다 — 본격 는 CH27.

흔한 실수와 점검

  • ctab_fname 미지정 → 색 컬럼을 둬도 Layout에 안 알리면 무시된다(CH21-L04).
  • 색 코드 → col_negative(빨강)·col_total(노랑) 등 상수 사용.
