WRITE로 문자열 출력

첫 출력 — WRITE로 화면에 글자를 찍어 개발 루프를 완성한다.

WRITE로 화면에 출력하기

작성 예정: 한 줄로 문자열을 출력하고 F8로 결과 화면 확인.

REPORT zhello.

WRITE 'Hello, ABAP!'.

여러 줄 출력과 줄바꿈

작성 예정: / 로 줄을 바꿔 출력.

WRITE 'first line'.
WRITE / 'second line'.

문자열 리터럴

작성 예정: 작은따옴표 '...' 문자 리터럴, 아직 변수는 없다(→ CH02 불편의 씨앗).

흔한 실수와 점검

작성 예정: 따옴표 짝 안 맞음, 마침표 누락.
