CH10 · SALV 1차 (간단 ALV)

기본 Function 표시와 Display 실행

표준 툴바를 켜고 display로 화면에 띄운다.

기능 켜고 띄우기

factory로 만든 객체에 표준 툴바(정렬·필터·합계·엑셀 등)를 켜고, display( )로 화면에 그린다.

표준 Function 켜기

lo_alv->get_functions( )->set_all( abap_true ).
  • get_functions( )가 기능 객체를 주고, set_all( abap_true )로 표준 버튼을 모두 켠다.
  • 이 한 줄로 정렬·필터·합계·레이아웃·엑셀 내보내기가 툴바에 생긴다.

화면에 표시 — display

lo_alv->display( ).
  • display( )가 실제로 ALV 화면을 그린다. 이게 없으면 아무것도 안 보인다.

흔한 실수와 점검

  • display 누락 → 객체만 만들고 안 띄움. 가장 흔한 "왜 안 보이지".
  • set_all 안 함 → 표는 떠도 툴바 기능이 비어 보인다.

조각이 다 모였다(factory → functions → display). 한 프로그램으로 이어 붙여 미니 리포트를 완성하자 → 다음 레슨.
