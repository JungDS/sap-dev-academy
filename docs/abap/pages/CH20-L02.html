CLASS zcl_booking_manager DEFINITION PUBLIC.
  PUBLIC SECTION.
    METHODS book IMPORTING iv_seats TYPE i.
    METHODS remaining RETURNING VALUE(rv) TYPE i.
  PRIVATE SECTION.
    DATA mv_capacity TYPE i.          " 인스턴스 속성
    DATA mv_booked   TYPE i.
ENDCLASS.
  • DATA — 속성(마다 따로). CLASS-DATA — 정적 속성(클래스 공용).
  • mv_ 접두어는 멤버 변수 관례.

공개 범위 — Visibility

섹션 누가 접근
PUBLIC SECTION 누구나(외부 포함)
PROTECTED SECTION 자신 + 받은 자식
PRIVATE SECTION 자신만
  • 외부에 보일 것()은 PUBLIC, 내부 데이터(잔여석 계산용)는 PRIVATE에 숨긴다(캡슐화).

흔한 실수와 점검

  • 데이터를 PUBLIC → 캡슐화가 깨진다. 데이터는 보통 PRIVATE, 접근은 로.
  • vs 정적 → 마다 다른 값=DATA, 공용=CLASS-DATA.

속성·메서드·범위를 정했다. 객체가 태어날 때 초기화하는 생성자 → 다음 레슨.
