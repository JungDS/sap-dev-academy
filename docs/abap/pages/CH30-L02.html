RFC Function Module 설계

외부에서 호출 가능한 함수 — Remote-Enabled FM.

다른 시스템이 내 함수를 부르게

(Remote Function Call)는 다른 시스템·외부 프로그램이 내 Function Module을 부르게 한다. SE37에서 함수를 Remote-Enabled로 만든다.

" 호출 측: 대상 시스템(RFC Destination)을 지정해 원격 호출
CALL FUNCTION 'Z_GET_BOOKINGS'
  DESTINATION 'TARGET_SYS'           " SM59에 등록된 연결
  EXPORTING  iv_concert = lv_id
  IMPORTING  et_booking = lt_booking
  EXCEPTIONS communication_failure = 1
             system_failure        = 2.
  • Remote-Enabled 함수는 파라미터에 제약(테이블·구조는 가능, 일부 타입 제한).
  • SM59 — RFC Destination(대상 시스템 연결) 등록.
  • 외부(Java·.NET 등)도 RFC로 ABAP 함수를 부른다.

흔한 실수와 점검

  • DESTINATION 누락 → 원격 호출엔 대상 연결이 필요. SM59 등록.
  • 통신 예외 미처리 → 네트워크 실패를 communication_failure로 대비.

원격 호출을 봤다. 이제 화면을 자동으로 채우는 옛 방식 — BDC → 다음 레슨.
