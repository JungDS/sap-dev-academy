개발 환경과 첫 프로그램

WRITE 심화 — 정렬·폭·색·구분선

출력을 보기 좋게 — 폭과 정렬로 칸을 맞추고, 색·구분선으로 강조한다.

Track 1Chapter 1Lesson 5입문

🖥️이번 Lesson에서 다루는 트랜잭션 코드

출력은 됐는데, 투박하다

앞 레슨에서 로 화면에 글자를 찍었다. 그런데 숫자를 여러 줄 출력해 보면 자릿수가 안 맞아 들쭉날쭉하고, 중요한 합계도 평범한 글자로 묻힌다. ABAP의 는 사실 출력 서식 옵션을 풍부하게 갖고 있다. 이 레슨에서 그걸 꺼내 쓴다.

🧾 리스트 출력은 결국 표처럼 줄을 맞춰 보여줄 때가 많다. 그래서 "폭과 정렬"이 첫 번째 도구다.

폭과 정렬 — 칸을 맞춘다

값을 정해진 폭(len) 안에 출력하고, 그 안에서 어디로 붙일지를 정렬로 정한다. 폭은 WRITE /(len)처럼 괄호로, 정렬은 키워드로 지정한다.

ABAP
1
2
3
WRITE: /(16) '정훈영'.                  " 폭 16, 기본(왼쪽)
WRITE: /(16) 1250000 RIGHT-JUSTIFIED.   " 폭 16, 오른쪽 정렬
WRITE: /(16) '합 계' CENTERED.          " 폭 16, 가운데
정렬 키워드 뜻 잘 쓰는 곳
(기본) / LEFT-JUSTIFIED 왼쪽 붙임 텍스트
RIGHT-JUSTIFIED 오른쪽 붙임 숫자(자릿수 정렬)
CENTERED 가운데 제목·라벨

직접 다뤄 보자

말보다 손이 빠르다. 아래에서 값과 옵션을 바꾸면 왼쪽에 코드가 만들어지고 오른쪽 리스트에 결과가 그려진다. 같은 값이라도 RIGHT-JUSTIFIED·COLOR·ULINE으로 얼마나 달라지는지 직접 보자.

🖱️ 예제 칩 "오른쪽 정렬 숫자" 와 "강조 색 텍스트" 를 번갈아 누르면, 숫자 정렬과 색 강조가 코드·출력에 어떻게 반영되는지 한눈에 비교된다.

색과 강조 — 눈에 띄게

합계나 경고처럼 튀어야 하는 값은 색과 강조로 표시한다.

ABAP
1
2
3
WRITE: /(16) 1250000 COLOR 5.              " 5 = 긍정(초록)
WRITE: /(16) '경고!'  COLOR 6 INTENSIFIED. " 6 = 부정(빨강) + 진하게
WRITE: /(16) '반전'   COLOR 4 INVERSE.      " 배경·글자 반전
  • COLOR n — 0~7번의 표준 리스트 색(예: 3 합계, 5 긍정, 6 부정).
  • INTENSIFIED — 더 진하게, INVERSE — 배경과 글자색을 뒤집는다.

💡 색 번호는 의미로 외우면 쉽다 — COLOR 5(초록)는 "좋음", COLOR 6(빨강)은 "문제". 위 플레이그라운드에서 COLOR를 바꿔 보라.

줄과 구분선 — 여백을 준다

빽빽한 출력은 읽기 어렵다. 빈 줄과 구분선으로 숨 쉴 공간을 준다.

ABAP
1
2
3
4
SKIP.            " 빈 줄 하나
SKIP 2.          " 빈 줄 둘
ULINE.           " 가로 구분선(밑줄)
WRITE: / '소계'.
  • SKIP / SKIP n — 빈 줄을 1개/n개 넣는다.
  • ULINE — 가로 구분선을 긋는다(표의 칸 구분에 자주 쓴다).

숫자·날짜 다듬기

숫자와 날짜는 전용 옵션으로 더 깔끔하게 만든다.

ABAP
1
2
3
WRITE: / lv_num   NO-ZERO.        " 앞의 의미 없는 0 숨김
WRITE: / lv_amt   DECIMALS 2.     " 소수 2자리로
WRITE: / sy-datum DD/MM/YYYY.     " 날짜 형식 지정

🔎 [선행 사용] 위 lv_num·lv_amt(변수)와 sy-datum(오늘 날짜)은 서식 효과를 보여주려고 미리 끌어다 쓴 것이다 — 변수는 CH02, sy-datum은 CH04에서 정식으로 배운다(지금은 분석·암기 불필요). 핵심은 날짜를 그냥 출력하면 20260619처럼 붙어 나오니 DD/MM/YYYY 형식을 주면 읽기 좋다는 서식 효과다.

흔한 실수와 점검

⚠️ 폭보다 긴 값 — 폭(len)보다 값이 길면 잘리거나 *로 표시된다. 폭을 넉넉히 준다.

⚠️ 숫자를 왼쪽 정렬 — 숫자는 보통 RIGHT-JUSTIFIED로 자릿수를 맞춰야 읽기 좋다.

⚠️ 색·강조 남발 — 다 강조하면 아무것도 강조가 안 된다. 정말 중요한 값에만.

정리

  • 는 값을 폭(len) 안에 넣고 정렬(LEFT·CENTERED·RIGHT-JUSTIFIED)로 칸을 맞춘다.
  • COLOR·INTENSIFIED·INVERSE 로 중요한 값을 강조한다.
  • SKIP(빈 줄)·ULINE(구분선)으로 읽기 좋게 띄운다.
  • 숫자·날짜는 NO-ZERO·DECIMALS·DD/MM/YYYY 같은 전용 옵션으로 다듬는다.
