Editable Grid ALV와 입력 검증

Editable Field Catalog 설정

ALV에서 직접 입력받게 — 편집 가능 컬럼 설정.

Track 2Chapter 28Lesson 1고급

보여주기에서 입력받기로

지금까지 ALV는 표시·였다. 이제 표 안에서 직접 값을 입력·수정받는다(좌석 수 조정 등). 먼저 어떤 컬럼을 편집 가능하게 할지 정한다.

ABAP
1
2
3
4
5
LOOP AT lt_fcat REFERENCE INTO DATA(lr).
  IF lr->fieldname = 'SEATS'.
    lr->edit = abap_true.        " 이 컬럼 편집 가능
  ENDIF.
ENDLOOP.

입력 이벤트 등록

편집한 값을 받으려면 그리드에 편집 를 등록해야 한다.

ABAP
1
2
go_grid->register_edit_event(
  i_event_id = cl_gui_alv_grid=>mc_evt_modified ).   " Enter/포커스 이동 시 발생
  • mc_evt_modified — 셀을 고치면 이벤트 발생.
  • mc_evt_enter — Enter 시 발생.

편집 모드 켜 보기

편집 모드를 토글하면 SEATS만 입력칸으로 바뀌고 키·표시 컬럼은 그대로인 걸 확인해 보라.

흔한 실수와 점검

  • edit만 켜고 미등록 → 값은 보이지만 프로그램이 변경을 못 받는다.
  • 전체 편집 → 키·계산 컬럼은 보통 읽기전용. 필요한 컬럼만 edit.

편집 가능하게 했다. 이제 값이 바뀌는 순간을 잡는 DATA_CHANGED → 다음 레슨.
