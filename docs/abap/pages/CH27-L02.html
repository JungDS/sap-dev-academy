Hotspot Click Event

셀을 링크처럼 — 한 번 클릭으로 이동.

한 번 클릭하는 링크

공연ID 컬럼을 링크처럼 만들어, 한 번 클릭하면 공연 상세로 보내고 싶다. 그게 Hotspot이다.

컬럼을 hotspot으로 — fieldcat

ABAP
1
2
3
4

LOOP AT lt_fcat REFERENCE INTO DATA(lr).
  IF lr->fieldname = 'CONCERT_ID'.
    lr->hotspot = abap_true.        " 링크 표시
  ENDIF.
ENDLOOP.

  IMPORTING e_row_id e_column_id.

METHOD on_hotspot.
  READ TABLE lt_booking INTO DATA(ls) INDEX e_row_id-index.
  " ls-concert_id 로 공연 상세 이동
ENDMETHOD.

직접 클릭해 보기

concert_id가 밑줄 링크로 보인다. 한 번 클릭하면 hotspot_click이 행·컬럼과 함께 실행되는 걸 확인해 보라(더블클릭 아님).

흔한 실수와 점검

  • hotspot 미지정 → 링크가 안 생긴다. fieldcat에서 hotspot.
  • double_click과 혼동 → hotspot은 한 번 클릭. 둘 다 두면 헷갈린다.

링크를 만들었다. 이제 표 위 툴바에 내 버튼을 추가하자 → 다음 레슨.
