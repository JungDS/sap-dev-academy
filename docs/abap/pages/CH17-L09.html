매진 회차를 빨갛게

"매진된 회차는 빨강"처럼 행 전체를 색으로 강조하면 한눈에 들어온다. 행 색은 각 행에 색 코드 컬럼을 두고, Layout에 그 컬럼 이름을 알려 주는 방식이다.

" 1) 출력 구조에 색 컬럼 추가(CHAR4) — 예: ROWCOLOR
" 2) 데이터 채울 때 색 코드 설정
LOOP AT lt_perf INTO ls_perf.
  IF ls_perf-seatsocc >= ls_perf-capacity.
    ls_perf-rowcolor = 'C610'.        " 빨강 계열 코드
    MODIFY lt_perf FROM ls_perf.
  ENDIF.
ENDLOOP.

" 3) Layout에 색 컬럼 이름 지정
ls_layout-info_fname = 'ROWCOLOR'.
  • 색 코드는 C + 색번호 + 강조 + 반전 형식(예: C610).
  • info_fname — 행 색을 담은 컬럼 이름을 Layout에 알려 준다.

🔶 셀 하나만 색을 다르게 하는 셀 색상과 (더블클릭 등)는 표시 제어 심화인 CH21에서 다룬다. 여기선 행 색상까지.

흔한 실수와 점검

  • info_fname 미지정 → 색 컬럼을 둬도 Layout에 안 알리면 색이 안 먹는다.
  • 색 코드 형식 → C로 시작하는 4자리 코드. 임의 RGB 아님.

화면에 박는 Grid ALV의 조각을 모두 배웠다(컨테이너→그리드→fieldcat→layout→display→ refresh→색). 이제 그 조각들을 하나의 실행 프로그램으로 모아 보자 → CH17-L10(종합 실습).
