보기 좋은 출력 — 양식

ALV·WRITE는 화면용이다. 송장·티켓처럼 인쇄·PDF 양식은 전용 도구로 만든다. 전통적 도구가 (SMARTFORMS)다.

구조

  • Form — 양식 전체. 안에 Page·Window.
  • Window — 양식 위 영역(메인/보조). 텍스트·테이블·그래픽 노드를 담는다.
  • Form Interface — 양식에 넘길 데이터(예매 정보)를 받는 .
  • 활성화하면 생성 함수 모듈(/1BCDWB/SF...)이 만들어진다.

" 양식의 생성 함수명을 얻어 호출(개념)
CALL FUNCTION 'SSF_FUNCTION_MODULE_NAME'
  EXPORTING formname = 'ZSF_TICKET'
  IMPORTING fm_name  = DATA(lv_fm).
CALL FUNCTION lv_fm
  EXPORTING is_booking = ls_booking.

흔한 실수와 점검

  • 생성 함수명 하드코딩 → 시스템마다 다르다. SSF_FUNCTION_MODULE_NAME으로 조회.
  • 미정의 → 양식에 데이터를 넘기려면 Form Interface 선언.

전통 양식을 봤다. 더 현대적인 Adobe Forms → 다음 레슨.
