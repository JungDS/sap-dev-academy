New Open SQL / Modern ABAP SQL

@ Host Variable과 Host Expression

ABAP 값을 SQL에 안전하게 넣는다 — @변수와 @( 식 ).

Track 1Chapter 19Lesson 2중급

@ — "이건 ABAP 값이다"

modern SQL에서 ABAP 변수·상수·필드를 SQL에 넣을 땐 앞에 **@**를 붙인다. 이를 host variable이라 한다.

ABAP
1
2
3
4
SELECT * FROM sflight
  WHERE carrid = @lv_carr
    AND fldate = @lv_date
  INTO TABLE @lt_flight.

@( ) — 식도 통째로

단순 변수뿐 아니라 식(계산·메서드 결과)도 @( ... )로 넣을 수 있다(host expression).

ABAP
1
2
3
SELECT * FROM sflight
  WHERE seatsmax > @( lv_base + 100 )
  INTO TABLE @lt_flight.
  • @lv_x — ABAP 변수 하나.
  • @( 식 ) — 계산식·상수 표현 등.

흔한 실수와 점검

  • 컬럼에 @ → DB 컬럼(carrid)엔 @를 안 붙인다. ABAP 값에만.
  • @( ) 괄호 누락 → 식은 반드시 @( ... )로 감싼다.

ABAP 값을 안전하게 넣었다. 이제 결과 받을 테이블도 인라인으로 선언하자 → 다음 레슨.
