SAPGUI 로그온과 화면 구성

SAP 시스템에 접속해 개발을 시작할 환경을 연다.

SAPGUI와 접속 환경

작성 예정: SAPGUI가 무엇이고 왜 필요한지, 클라이언트/서버 관계 한눈에.

시스템에 로그온하기

작성 예정: 접속 항목 등록 → Client/User/Password/Language 입력 → 로그온.

SAP Easy Access 둘러보기

작성 예정: 로그온 후 첫 화면, 즐겨찾기, 명령창 위치.

흔한 실수와 점검

작성 예정: 잘못된 Client/Language, 비밀번호 초기화 등.
