컬럼을 세밀하게

CH17의 를 다시 꺼내, 컬럼별 속성을 더 세밀하게 준다.

LOOP AT lt_fcat REFERENCE INTO DATA(lr_fc).
  CASE lr_fc->fieldname.
    WHEN 'MANDT'.   lr_fc->no_out   = abap_true.   " 숨김
    WHEN 'SEATS'.   lr_fc->do_sum   = abap_true.   " 합계
    WHEN 'STATUS'.  lr_fc->just     = 'C'.         " 가운데 정렬
  ENDCASE.
ENDLOOP.
  • no_out 숨김 · do_sum 합계 · just(L/C/R) 정렬 · key 키컬럼 고정.
  • edit = abap_true로 편집 가능 표시도 가능(본격 편집은 CH28).

흔한 실수와 점검

  • 합계 불가 타입 → 문자 컬럼엔 do_sum이 안 먹는다(숫자만).
  • 숨김 vs 미포함 → no_out은 카탈로그엔 있고 화면에만 숨김(다시 표시 가능).
