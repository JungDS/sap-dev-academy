SELECT-OPTIONS와 Range Table

Multiple Selection과 Include/Exclude

다중 선택 팝업 — 여러 값·범위와 포함/제외(녹색/빨강).

Track 1Chapter 11Lesson 4중급

여러 값, 그리고 제외

우측 화살표 버튼을 누르면 다중 선택 팝업이 뜬다. 여기서 "여러 값", "여러 범위", 그리고 제외(Exclude) 까지 지정한다.

팝업의 탭

  • Single values (포함) — 개별 값 여러 개.
  • Ranges (포함) — 여러 범위.
  • Exclude single values / ranges — 제외할 값/범위.

포함은 의 SIGN = 'I', 제외는 SIGN = 'E'로 들어간다(녹색/빨강 아이콘으로 구분).

동작 규칙(직관)

  • 포함(I) 조건들 중 하나라도 맞고(OR),
  • 제외(E) 조건에 걸리지 않으면 통과.

예: "20~40 포함, 단 30 제외" → I BT 20 40 + E EQ 30.

흔한 실수와 점검

  • 제외만 지정 → 포함이 없으면 결과가 비거나 의도와 다를 수 있다. 포함과 함께.
  • 녹색/빨강 의미 → 녹색=포함(I), 빨강=제외(E).

포함·제외까지 다뤘다. 각 조건의 비교 방식(OPTION) 을 정확히 알면 코드로도 조작할 수 있다 → EQ/BT/CP(다음 레슨).
