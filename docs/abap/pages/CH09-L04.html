Local Class 정의와 Method 호출

현대 모듈화의 중심 — 로컬 클래스와 메서드(맛보기).

Track 1Chapter 9Lesson 4초급

현대의 중심 — 클래스

새 ABAP은 (클래스)와 (메서드)로 모듈화한다. 여기선 맛보기만 한다(본격 OO 설계는 CH19). 프로그램 안에 정의하는 로컬 클래스부터.

정의 + 구현

CLASS lcl_calc DEFINITION.
  PUBLIC SECTION.
    METHODS add_tax
      IMPORTING iv_amount        TYPE p
      RETURNING VALUE(rv_result) TYPE p.
ENDCLASS.

CLASS lcl_calc IMPLEMENTATION.
  METHOD add_tax.
    rv_result = iv_amount * '1.1'.
  ENDMETHOD.
ENDCLASS.
  • DEFINITION — 무엇이 있는지(선언). IMPLEMENTATION — 실제 코드.
  • IMPORTING(입력) / RETURNING(반환) 등으로 파라미터를 선언.

객체 만들고 메서드 호출

START-OF-SELECTION.
  DATA(lo_calc) = NEW lcl_calc( ).
  DATA(lv_out)  = lo_calc->add_tax( 1000 ).
  WRITE lv_out.
  • NEW로 객체(인스턴스) 생성, ->로 호출.
  • (DATA(...) 인라인 선언은 현대 구문 — CH17에서 정식으로.)

흔한 실수와 점검

  • DEFINITION만, IMPLEMENTATION 없음 → 메서드 본문이 없어 오류.
  • 정적 vs 인스턴스 → 인스턴스 메서드는 객체(->)로 호출. 정적은 =>(CH19).

로컬 클래스는 그 프로그램 안에만 산다. 여러 곳에서 쓰는 전역 클래스는? → 다음 레슨.
