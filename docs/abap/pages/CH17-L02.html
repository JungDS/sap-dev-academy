Grid ALV 기초

CL_GUI_ALV_GRID 생성

컨테이너 위에 ALV 그리드 객체를 만든다.

Track 1Chapter 17Lesson 2중급

컨테이너 위에 그리드를 얹는다

는 화면 안에 박는 강력한 표 컨트롤이다. 만들 때 어느 컨테이너에 들어갈지를 알려 준다.

ABAP
1
2
3
4
DATA go_grid TYPE REF TO cl_gui_alv_grid.

CREATE OBJECT go_grid
  EXPORTING i_parent = go_cont.    " 앞 레슨의 컨테이너
  • i_parent — 그리드를 담을 부모 컨테이너(go_cont).
  • 아직 데이터는 안 보인다 — 무엇을(내부 테이블) 어떻게(Field Catalog) 보여줄지 더 준비해야 한다(다음 레슨들).

SALV와 무엇이 다른가

SALV(CH11) Grid ALV(CH17)
코드량 적음(factory) 많음(컨테이너·fieldcat 등)
화면 박기 전체 화면 내 화면 영역에
제어·편집 제한적·읽기전용 세밀·편집 가능(CH28)

흔한 실수와 점검

  • 부모 컨테이너 없음 → i_parent에 유효한 컨테이너를 넘겨야 한다.
  • 그리드만 만들고 끝 → 데이터·Field Catalog·display까지 해야 표가 보인다.

그리드 객체는 만들었다. 이제 보여줄 데이터(내부 테이블)를 준비하자 → 다음 레슨.
