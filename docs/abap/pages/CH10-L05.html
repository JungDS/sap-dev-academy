CH10 · SALV 1차 (간단 ALV)

SALV 1차 범위와 심화 항목 분리

지금 다루는 SALV의 범위와, 뒤로 미루는 심화의 경계.

어디까지가 'SALV 1차'인가

는 더 많은 걸 할 수 있지만, 지금(1차)은 읽기전용 기본 표시 + 표준 기능 까지만 다룬다. 나선형 학습이라 심화는 명확히 뒤로 미룬다.

지금(1차) 범위

  • factory로 내부 테이블 표시
  • set_all로 표준 툴바(정렬·필터·합계·엑셀)
  • display로 출력

뒤로 미루는 심화

항목 어디서
컬럼 제목/너비/통화·수량 포맷 세밀 제어 SALV 표시 심화(CH20)
셀 색·Hotspot·줄 색 CH20
더블클릭·툴바 이벤트 ALV 이벤트(CH26)
셀 편집 Editable Grid ALV(CH27)
화면에 박는 컨테이너형 Grid ALV CH16

왜 나눴나

처음부터 모든 ALV 기능을 던지면 압도된다. "결과를 보기 좋게"라는 지금의 불편만 해결하고, 색·편집·이벤트는 각자의 불편이 생길 때 배운다.

흔한 오해와 점검

  • SALV로 다 하려 함 → 편집·복잡 이벤트는 Grid ALV. 도구를 구분.
  • 심화를 지금 → 나선형. 1차는 표시까지.

이제 결과를 표로 보여줄 수 있다. 그런데 입력 조건이 단일 값(PARAMETERS) 뿐이라 범위·다중 조건 조회가 답답하다 → CH11(SELECT-OPTIONS).
