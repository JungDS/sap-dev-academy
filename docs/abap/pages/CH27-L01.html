ALV 고급 Event 응용

Double Click Event

행을 더블클릭하면 상세로 — ALV 이벤트의 시작.

표를 클릭하면 반응하게

CH21까지 ALV는 보여주기만 했다. 이제 사용자가 행을 더블클릭하면 상세 화면으로 가는 식의 상호작용을 넣는다. CH20-L09의 OO (SET HANDLER)가 여기 쓰인다.

CLASS lcl_handler DEFINITION.
  PUBLIC SECTION.
    METHODS on_double_click
      FOR EVENT double_click OF cl_gui_alv_grid
      IMPORTING e_row e_column.
ENDCLASS.

CLASS lcl_handler IMPLEMENTATION.
  METHOD on_double_click.
    READ TABLE lt_booking INTO DATA(ls) INDEX e_row-index.
    IF sy-subrc = 0.
      " 예매 상세 화면 호출
    ENDIF.
  ENDMETHOD.
ENDCLASS.

연결 — SET HANDLER

ABAP
1
2
DATA(lo_h) = NEW lcl_handler( ).
SET HANDLER lo_h->on_double_click FOR go_grid.
  • e_row-index — 클릭한 행 번호. e_column-fieldname — 클릭한 컬럼.

직접 더블클릭해 보기

아래 ALV 표의 행을 더블클릭하면 핸들러가 받는 e_row-index·e_column이 로그에 찍힌다.

흔한 실수와 점검

  • SET HANDLER 누락 → 더블클릭해도 반응 없음(CH20-L09).
  • vs 키 → e_row-index로 내부 테이블 행을 READ.

더블클릭에 반응했다. 이제 셀을 링크처럼 만드는 Hotspot → 다음 레슨.
