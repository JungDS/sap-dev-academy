셋 중 무엇을 쓰나

(FORM) · · (메서드) — 도구가 셋이다. 상황별로 정리한다.

한눈에

도구 범위 권장도 언제
(FORM) 프로그램 내부 ▲ 레거시 기존 코드 유지보수
전역 재사용 ○ 절차적 재사용, RFC(원격), 표준 BAPI
/ Method 전역 재사용 ◎ 권장 신규 개발 전반(현대 표준)

원칙

  • 신규 코드는 클래스(메서드) 로 — 현대 ABAP의 기본.
  • 함수 모듈은 여전히 유효: 원격 호출(RFC), 다수의 표준 인터페이스가 FM.
  • 서브루틴(FORM) 은 새로 만들지 말 것. 읽고 유지보수하는 용도로만 안다.

흔한 실수와 점검

  • 신규에 FORM 사용 → 지양. 메서드로.
  • 무조건 클래스 → RFC가 필요하면 함수 모듈(또는 RFC 가능 FM)을 쓴다.

이제 코드를 재사용 단위로 묶을 수 있다. 배운 를 콘서트앱에 적용해 잔여석·예매 판정 로직을 만들어 보자 → CH10-L07(실습).
