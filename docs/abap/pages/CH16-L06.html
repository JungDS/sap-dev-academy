화면 위 메뉴·툴바

화면 상단의 메뉴바·툴바·기능키를 묶은 것이 (PF-STATUS)다. SE41(또는 Screen Painter)에서 만들고, 어떤 버튼을 둘지·각 버튼의 FctCode를 정한다.

ABAP

MODULE status_0100 OUTPUT.
  SET PF-STATUS 'ST100'.     " 이 화면에 쓸 GUI Status
  SET TITLEBAR  'TB100'.     " 제목 표시줄
ENDMODULE.
  • SET PF-STATUS로 PBO에서 상태를 켜면, 그 버튼들이 눌릴 때 FctCode가 OK_CODE로 온다.
  • SET TITLEBAR로 제목을 지정한다(텍스트는 변수로 채울 수도 있음 WITH).

버튼을 동적으로 숨기기

특정 상황에서 일부 버튼만 감추려면 SET PF-STATUS 'ST100' EXCLUDING lt_fcode.처럼 제외 목록을 넘긴다.

흔한 실수와 점검

  • PF-STATUS 미설정 → 표준 버튼만 떠 내 기능키가 없다. PBO에서 SET PF-STATUS.
  • FctCode 불일치 → Status의 버튼 코드와 CASE ok_code의 값이 같아야 한다.

화면·입력·메뉴까지 갖췄다. 이제 화면 안에 ALV·트리 같은 컨트롤을 박는 자리, Custom Control을 보자 → 다음 레슨.
