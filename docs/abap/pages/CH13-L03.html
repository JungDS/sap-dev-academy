Help View와 Search Help 연결

여러 테이블에서 F4 값을 뽑는 소스 — Help View.

F4 소스가 여러 테이블이면

CH08의 는 보통 한 테이블/뷰에서 값을 뽑는다. F4 목록을 여러 테이블을 조인해 만들고 싶다면, 그 조인 소스로 Help View를 쓴다.

Help View = Search Help의 조인 소스

  • Foreign Key 관계로 테이블들을 조인한 F4 전용 읽기 뷰.
  • 검색 도움말()의 Selection Method로 지정한다.
ZHV_PERSON (Help View: person ⨝ dept)
   ↑ Selection Method
ZSH_PERSON (Search Help) → 필드/Data Element에 부착 → 화면 F4

흔한 실수와 점검

  • Help View를 일반 조회에 → 주 용도는 F4 소스. 일반 조회는 .
  • 조인 키 부정확 → F4 목록이 엉뚱해진다. FK·조인 조건 확인.

조회·도움말 뷰를 봤다. 이제 데이터를 입력·수정하는 뷰다 → Maintenance View(다음 레슨).
