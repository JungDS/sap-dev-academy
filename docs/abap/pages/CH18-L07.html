" classic (CH10-L07)
DATA: ls_book TYPE zbooking, lv_sum TYPE i.
LOOP AT lt_book INTO ls_book.
  lv_sum = lv_sum + ls_book-seats.
ENDLOOP.

" modern (CH18)
DATA lv_total TYPE i.
LOOP AT lt_book INTO DATA(ls).      " 인라인 선언
  lv_total += ls-seats.            " 복합 대입
ENDLOOP.

구조체도 한 식으로 채운다.

DATA(ls_new) = VALUE zbooking(
  concert_id = 'C001' perf_no = '01' customer = '정훈영' seats = 2 status = 'N' ).

한 줄을 키로 집는다(Table Expression).

ABAP
1
DATA(ls_hit) = lt_book[ booking_id = lv_id ].   " READ TABLE 대신

직접 해 볼 과제

  1. CH10-L07 get_remaining의 LOOP를 인라인 DATA·+=로 바꾼다.
  2. 새 예매 구조체를 VALUE로 한 식에 채운다.
  3. (도전) READ TABLE을 Table Expression lt[ ... ]로 바꾸고 line_exists( )로 보호.

흔한 실수와 점검

  • SELECT에 인라인 → SELECT의 @DATA는 CH19. 여기선 LOOP/READ 등 SQL 밖에서만.
  • table expr 없는 행 → 없으면 예외. line_exists( )로 확인(CH18-L04).

ABAP 코드를 모던으로 다듬었다. 이제 SELECT 자체를 모던 SQL로 → CH19.
