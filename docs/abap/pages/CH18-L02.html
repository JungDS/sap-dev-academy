Modern ABAP Syntax

VALUE Constructor Expression

구조체·내부 테이블을 한 식으로 채운다 — VALUE

Track 1Chapter 18Lesson 2중급

한 줄 한 줄 채우기 — 번거롭다

classic에서는 구조체 필드를 하나씩 대입하고, 테이블은 APPEND를 여러 번 했다. 모던 VALUE #( )는 한 식으로 채운다.

구조체 채우기

ABAP
1
2
3
4
5
6
" classic
DATA ls_line TYPE ty_line.
ls_line-dan = 2. ls_line-mul = 3. ls_line-result = 6.

" modern
DATA(ls_line) = VALUE ty_line( dan = 2 mul = 3 result = 6 ).

내부 테이블 채우기

각 ( ... )가 한 행이다.

ABAP
1
2
3
4
DATA(lt_gugu) = VALUE ty_line_tab(
  ( dan = 2 mul = 1 result = 2 )
  ( dan = 2 mul = 2 result = 4 )
  ( dan = 2 mul = 3 result = 6 ) ).

기존에 더하기 — BASE

기존 테이블을 유지하면서 행을 더하려면 BASE를 쓴다.

ABAP
1
lt_gugu = VALUE #( BASE lt_gugu ( dan = 3 mul = 1 result = 3 ) ).

흔한 실수와 점검

  • 타입 생략 VALUE #( ) → 좌변 타입이 분명할 때만 #. 아니면 타입명을 적는다.
  • BASE 빠뜨림 → VALUE만 쓰면 기존 내용이 사라진다. 누적은 BASE.

값을 한 식으로 채웠다. 이제 구조가 다른 둘 사이를 이름으로 매핑하는 CORRESPONDING → 다음 레슨.
