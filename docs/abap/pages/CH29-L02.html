표준 코드 라인 사이에 끼우기

Enhancement Framework은 표준 코드 특정 지점에 내 코드를 추가(Point)하거나 교체(Section)하게 한다. 표준 소스는 그대로 두고 확장만 얹는다.

ENHANCEMENT zenh_my_logic.    " 표준이 마련한 지점에 끼운 내 코드
  IF gv_amount > 10000.
    " 추가 처리
  ENDIF.
ENDENHANCEMENT.
종류 동작
Enhancement Point 그 자리에 코드 추가
Enhancement Section 표준 블록을 내 것으로 교체
  • ·함수의 시작/끝엔 Implicit Enhancement(아래 L04)가 늘 존재한다.

라인 사이에 끼워 보기

표준 코드 라인 사이의 Enhancement Point에 ENHANCEMENT 블록을 넣고 빼 보라 — 표준 라인은 그대로다.

흔한 실수와 점검

  • Section 남용 → 표준 블록 교체는 위험(표준 변경 시 어긋남). 가능하면 Point.
  • 활성화 → Enhancement Implementation을 활성화해야 동작.

코드에 끼웠다. 이제 가장 권장되는 지향 확장 — BAdI → 다음 레슨.
