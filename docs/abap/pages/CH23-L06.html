Service Definition / Service Binding

앱을 OData 서비스로 노출한다.

Track 1Chapter 23Lesson 6고급

바깥으로 노출하기

만든 RAP 앱을 Fiori·외부가 쓰려면 서비스로 노출해야 한다. 두 단계다.

① Service Definition — 무엇을 노출할지.

ABAP
1
2
3
4
define service ZUI_Booking {
  expose ZC_Booking as Booking;
  expose ZI_Concert as Concert;
}

② Service Binding — 어떤 프로토콜로(OData V2/V4·UI/Web API).

  • ADT에서 Service Binding을 만들고 Activate → OData 서비스 생성.
  • "Preview"로 Fiori Elements 화면을 바로 띄워 볼 수 있다.

흔한 실수와 점검

  • expose 누락 → Service Definition에 넣은 것만 노출된다.
  • binding 미활성화 → Activate해야 URL이 생기고 호출 가능.

서비스로 노출했다. 이제 비즈니스 규칙 — 검증·결정·액션을 보자 → 다음 레슨.
