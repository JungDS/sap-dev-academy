lo->get_display_settings( )>set_striped_pattern( abap_true ).   " 줄무늬

lo_cols->set_optimize( abap_true ).                              " 너비 최적화

Variant 저장

ABAP
1
2
lo->get_layout( )>set_key( VALUE #( report = sy-repid ) ).
lo->get_layout( )>set_save_restriction( ).                       " 사용자 레이아웃 저장 허용

흔한 실수와 점검

  • 컬럼명 오타 → get_column( )은 존재하는 컬럼명이라야.
  • 레이아웃 키 누락 → Variant 저장하려면 set_key로 식별.

SALV를 다듬었다. 이제 Grid ALV의 컬럼을 더 세밀히 제어하자 → 다음 레슨.
