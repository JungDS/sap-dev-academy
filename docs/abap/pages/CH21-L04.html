Deep Structure 기반 Cell Color

셀 단위 색 — 행 구조에 색 테이블을 품는다.

Track 1Chapter 21Lesson 4고급

행이 아니라 셀 하나만

CH17에서 행 전체 색을 봤다. 이제 특정 셀 하나(예: 매진 회차의 잔여석 칸만 빨강)에 색을 준다. 방법은 행 구조에 색 정보 테이블(LVC_T_SCOL)을 컬럼으로 품는 것 — CH06의 가 여기 쓰인다.

TYPES: BEGIN OF ty_perf,
         concert_id TYPE zconcert-concert_id,
         seatsocc   TYPE i,
         capacity   TYPE i,
         cellcolors TYPE lvc_t_scol,        " 셀 색 정보(테이블 in 행)
       END OF ty_perf.

각 행을 채울 때, 색을 줄 셀을 지정한다.

LOOP AT lt_perf ASSIGNING FIELD-SYMBOL(<perf>).
  IF <perf>seatsocc >= <perf>capacity.
    <perf>cellcolors = VALUE #( ( fname = 'SEATSOCC'
                                   color-col = col_negative ) ).  " 빨강 계열
  ENDIF.
ENDLOOP.

ls_layout-ctab_fname = 'CELLCOLORS'.     " 셀색 컬럼 이름을 Layout에 알림
  • LVC_T_SCOL 한 줄 = (어느 컬럼 fname + 색 color).
  • ctab_fname — 셀 색 테이블 컬럼 이름을 Layout에 지정.

흔한 실수와 점검

  • ctab_fname 미지정 → 색 컬럼을 둬도 Layout에 안 알리면 무시된다.
  • fname 오타 → 색 줄 컬럼명이 fieldcat과 정확히 같아야.

셀에 색을 줬다. 이제 셀 모양·동작(편집·버튼)을 정하는 Cell Style → 다음 레슨.
