SALV/Grid ALV 표시 제어 심화

Row / Column / Cell Color 선택 기준

상황에 맞춰 행·열·셀 단위로 컬러를 설정하는 기준을 소개한다.

Track 1Chapter 21Lesson 6고급

색을 줄 세 단위

단위 방법 언제
행 색 info_fname(행에 색코드 컬럼) "이 줄 전체가 매진" — 행 상태 강조
컬럼 색 fieldcat emphasize "이 컬럼이 중요" — 특정 열 강조
셀 색 ctab_fname(LVC_T_SCOL) "이 칸만 문제" — 셀 단위 강조

고르는 기준

  • 행 전체의 상태(매진/취소) → 행 색(가장 단순).
  • 특정 컬럼을 늘 강조(예: 합계 컬럼) → 컬럼 색.
  • 조건부 한 칸(잔여석이 위험할 때 그 칸만) → 셀 색(가장 세밀, deep 구조 필요).

🎯 콘서트 앱: 매진 회차는 행 빨강, 잔여 적은 회차는 잔여석 셀 노랑 — 단위를 섞어 직관적으로.

흔한 실수와 점검

  • 과한 색 → 다 칠하면 강조가 무의미. 정말 중요한 것만.
  • 셀 색 남용 → deep 구조·계산 비용. 행/컬럼으로 충분하면 그쪽을.

색을 자유롭게 줬다. 마지막으로 데이터를 갱신해도 보던 상태를 지키는 Stable Refresh → 다음 레슨.
