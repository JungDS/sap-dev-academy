" --- classic ---
DATA ls TYPE ty_line.
DATA lt TYPE ty_line_tab.
ls-dan = 2. ls-mul = 1. ls-result = 2.
APPEND ls TO lt.

" --- modern ---
DATA(lt) = VALUE ty_line_tab( ( dan = 2 mul = 1 result = 2 ) ).

복합 대입 — += -= *= /=

lv = lv + 10을 더 짧게 쓴다(CH04에서 예고 없이 미뤄 둔 그 문법).

언제까지 모던, 언제 classic

  • 새 코드는 모던 우선(짧고 안전).
  • 기존 classic 코드를 읽고 유지보수할 줄도 알아야 한다(둘 다 현장에 있다).

흔한 실수와 점검

  • 과한 한 줄 → 너무 많은 식을 한 줄에 몰면 오히려 안 읽힌다. 적당히 끊는다.
  • SELECT 모던 혼동 → SELECT 안의 modern 구문(콤마·escape)은 다음 챕터(CH19) 다.
