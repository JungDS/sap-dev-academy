CH14 · Report Event·Selection Screen 심화

AT SELECTION-SCREEN OUTPUT 동적 화면 제어

화면 그리기 직전(PBO) — 필드를 동적으로 숨김/잠금.

화면을 상황에 맞게

"옵션 A를 고르면 B 입력칸을 숨긴다"처럼 화면을 동적으로 바꾸고 싶다. 화면이 그려지기 직전(PBO) 마다 실행되는 AT SELECTION-SCREEN OUTPUT에서 한다.

LOOP AT SCREEN

AT SELECTION-SCREEN OUTPUT.
  LOOP AT SCREEN.
    IF screen-name = 'P_SECRET'.
      screen-active = '0'.        " 숨김 (0 = 비활성)
      MODIFY SCREEN.
    ENDIF.
  ENDLOOP.
  • LOOP AT SCREEN으로 화면 요소를 돌며 screen-active/screen-input 등을 바꾸고 MODIFY SCREEN으로 적용한다.
  • 이 이벤트는 화면을 그릴 때마다(매 PBO) 실행된다.

흔한 실수와 점검

  • MODIFY SCREEN 누락 → 바꿔도 반영 안 됨. 꼭 MODIFY SCREEN.
  • 여기서 무거운 로직 → 매번 실행되니 가볍게. 조회는 START-OF-SELECTION.

화면 모양은 제어했다. 이제 사용자가 실행을 누른 순간 입력을 검증하자 → AT SELECTION-SCREEN(다음 레슨).
