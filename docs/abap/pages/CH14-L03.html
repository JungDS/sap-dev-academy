Help View와 Search Help 연결

여러 테이블에서 F4 값을 뽑는 소스 — Help View.

Track 1Chapter 14Lesson 3중급

F4 소스가 여러 테이블이면

CH09의 는 보통 한 테이블/뷰에서 값을 뽑는다. F4 목록을 여러 테이블을 해 만들고 싶다면, 그 조인 소스로 Help View를 쓴다.

Help View = Search Help의 조인 소스

  • Foreign Key 관계로 테이블들을 한 F4 전용 읽기 뷰.
  • 검색 도움말()의 Selection Method로 지정한다.
