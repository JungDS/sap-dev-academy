CH11 · SELECT-OPTIONS와 Range Table

EQ / BT / CP 옵션 이해

비교 방식 OPTION 값 — EQ·BT·CP 등.

OPTION — 비교 방식

의 OPTION 컬럼은 어떻게 비교할지를 정한다. 자주 쓰는 값들:

OPTION 뜻 쓰임
EQ 같음(=) 단일 값
NE 다름(<>)
BT 범위(between) LOW~HIGH
NB 범위 밖
GT/GE 초과/이상
LT/LE 미만/이하
CP 패턴 포함(contains pattern) *,+ 와일드카드
NP 패턴 불포함

SIGN과 함께

SIGN(I/E) × OPTION(EQ/BT/CP…) 조합으로 조건이 결정된다.

  • I EQ 'KR' — 'KR' 포함.
  • I BT 20 40 — 20~40 포함.
  • I CP 'A*' — 'A'로 시작하는 것 포함.
  • E EQ 'XX' — 'XX' 제외.

CP 패턴 와일드카드

  • * — 임의의 여러 문자. + — 임의의 한 문자.
  • 예: CP 'A*'(A로 시작), CP '*son'(son으로 끝).

흔한 실수와 점검

  • CP에 일반 비교 기대 → CP는 패턴이라 */+가 와일드카드. 정확매칭은 EQ.
  • BT인데 HIGH 빔 → 범위는 LOW·HIGH 둘 다.

값을 화면 없이 코드로 직접 넣고 싶을 때도 있다 → Selection Table 조작(다음 레슨).
