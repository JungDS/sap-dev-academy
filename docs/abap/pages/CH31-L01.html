시스템 간 표준 메시지

(Intermediate Document)은 SAP 표준의 메시지 컨테이너다. 주문·송장 같은 업무 문서를 시스템 간에 비동기로 주고받는다(EDI·B2B).

3층 구조

부분 내용
Control Record(EDIDC) 발신/수신·메시지 타입·방향
Data Records(EDIDD) 실제 데이터 — Segment들의 묶음
Status Records(EDIDS) 처리 상태 이력
  • Basic Type(예: ORDERS05) — 의 구조 정의(어떤 Segment로 구성).
  • WE02/WE05 — 모니터, WE60 — 구조 문서.
