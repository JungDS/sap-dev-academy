GROUP BY와 Aggregate

묶어서 세고 합산한다 — GROUP BY와 집계 함수.

Track 1Chapter 12Lesson 3중급

묶어서 요약하기

"부서별 인원수", "부서별 급여합계"처럼 그룹마다 요약하고 싶다. GROUP BY와 (집계 함수)다.

GROUP BY + 집계 (classic)

ABAP

SELECT dept_id COUNT( * ) AS cnt SUM( salary ) AS total
  FROM ztperson
  INTO CORRESPONDING FIELDS OF TABLE lt_sum
  GROUP BY dept_id.
  • 집계 함수: COUNT( * ), SUM( ), MAX( ), MIN( ), AVG( ).
  • 그룹 키가 아닌 컬럼은 집계 함수로 감싸야 한다(GROUP BY에 없는 평컬럼 금지).
  • 결과 구조에 cnt, total 같은 별칭 필드를 둔다.

결과 받기

INTO CORRESPONDING FIELDS OF TABLE로 별칭과 같은 이름 필드에 담는다(합계가 커질 수 있으니 타입을 충분히 크게).

흔한 실수와 점검

  • 그룹키 아닌 평컬럼 SELECT → 오류. 집계로 감싸거나 GROUP BY에 추가.
  • 합계 오버플로 → SUM 결과 타입(P 길이)을 충분히.
  • 콤마/@ → classic이라 공백 구분, @ 없이.

그룹 요약은 됐다. "인원 5명 이상인 부서만"처럼 집계 결과로 필터하려면? → HAVING(다음 레슨).
