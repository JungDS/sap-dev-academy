CH01 · 개발 환경과 첫 프로그램

T-code와 SE38 첫 실행

명령창과 SE38(ABAP Editor)로 첫 프로그램을 만들고 $TMP(Local Object)에 저장한다.

T-code: 어디든 바로 가는 주소

작성 예정: 메뉴 트리 대신 명령창에 를 입력해 화면으로 직행.

SE38 ABAP Editor 열기

작성 예정: /nSE38 입력 → ABAP Editor 진입.

첫 프로그램 생성하기

작성 예정: 프로그램명 규칙(Z/Y 네임스페이스), 타입 = 실행 프로그램.

$TMP에 저장하기 (Local Object)

작성 예정: 저장 시 패키지를 묻는 다이얼로그 → Local Object 선택 → $TMP에 보관(이송요청 없이 바로). 지금은 이게 가장 간단 — 단, 이송·공유가 안 되는 한계가 있다(→ L05).

활성화와 실행 (개발 루프)

작성 예정: 작성 → 저장 → 활성화(Ctrl+F3) → 실행(F8). 이 루프를 몸에 익힌다.

흔한 실수와 점검

작성 예정: 네임스페이스 위반, 미활성화 상태로 실행 시도.
