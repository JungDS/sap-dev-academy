Cell Style 기반 입력 제어

셀마다 편집 가능/불가를 동적으로 — Cell Style.

행마다 다른 입력 규칙

"매진 회차는 좌석 수 수정 불가, 그 외는 가능"처럼 셀마다 편집 여부가 다르다. CH21-L05의 Cell Style(LVC_T_STYL)로 동적 제어한다.

DATA lv_style TYPE i.

LOOP AT lt_perf ASSIGNING FIELD-SYMBOL(<p>).
  IF <p>seatsocc >= <p>capacity.
    lv_style = cl_gui_alv_grid=>mc_style_disabled.   " 매진 → 읽기전용
  ELSE.
    lv_style = cl_gui_alv_grid=>mc_style_enabled.    " 그 외 → 편집
  ENDIF.
  <p>cellstyles = VALUE #( ( fieldname = 'SEATS' style = lv_style ) ).
ENDLOOP.

ls_layout-stylefname = 'CELLSTYLES'.

매진 토글로 확인

행을 매진으로 바꾸면 그 행의 SEATS 셀만 읽기전용으로 잠기는 걸 확인해 보라(같은 컬럼, 셀마다 다른 스타일).

흔한 실수와 점검

  • fieldcat edit과 충돌 → 전체는 fieldcat edit, 는 cell style로 덮어쓴다.
  • 갱신 후 style 재설정 → 데이터가 바뀌면 스타일도 다시 계산.

입력 가능 여부를 셀별로 제어했다. 이제 입력값을 검증하고 오류를 보여주자 → 다음 레슨.
