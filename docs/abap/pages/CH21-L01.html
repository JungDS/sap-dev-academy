SALV를 코드로 더 다루기

CH11에서 SALV를 띄우기만 했다. 이제 OO(CH20)를 배웠으니, SALV 의 로 정렬·필터·툴바 기능을 코드로 제어한다.

DATA(lo_salv) = cl_salv_table=>factory(
  IMPORTING r_salv_table = DATA(lo) CHANGING t_table = lt_perf ).

lo->get_functions( )>set_all( abap_true ).          " 표준 기능 켜기
lo->get_sorts( )>add_sort( 'PERF_DATE' ).            " 날짜로 정렬
lo->display( ).
  • get_functions( ) — 툴바 표준 기능(정렬·합계·엑셀 등).
  • get_sorts( )->add_sort( ) — 기본 정렬 지정.
  • 필터·도 get_filters( )·get_aggregations( )로.

흔한 실수와 점검

  • display 전에 설정 → 정렬·기능은 display( ) 전에 지정한다.
  • 존재하지 않는 필드 → add_sort의 필드명은 정확한 컬럼명.

기능을 제어했다. 이제 표시 레이아웃·Variant를 더 깊이 다루자 → 다음 레슨.
