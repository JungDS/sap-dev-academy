ALV 툴바에 내 버튼을 추가한다.

표준 버튼만으론 부족

"선택한 예매 취소" 같은 내 기능 버튼을 ALV 툴바에 추가한다. toolbar 에서 버튼을 끼워 넣는다.

METHODS on_toolbar
  FOR EVENT toolbar OF cl_gui_alv_grid
  IMPORTING e_object.

METHOD on_toolbar.
  APPEND VALUE #( function  = 'ZCANCEL'
                  icon      = icon_delete
                  text      = '예매 취소'
                  butn_type = 0 ) TO e_object->mt_toolbar.
ENDMETHOD.
  • e_object->mt_toolbar에 버튼을 추가.
  • function = 'ZCANCEL' — 눌렀을 때 올 명령(다음 레슨 user_command).
  • 버튼 추가 후 go_grid->set_toolbar_interactive( )로 갱신하기도 한다.

흔한 실수와 점검

  • function 코드 누락 → 눌러도 무슨 버튼인지 모른다. 고유 코드를.
  • 갱신 안 됨 → 동적으로 바꾸면 toolbar를 다시 그리도록.

버튼을 달았다. 이제 그 버튼이 눌렸을 때를 처리하자 → USER_COMMAND(다음 레슨).
