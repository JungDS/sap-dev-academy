Report Event·Selection Screen 심화

INITIALIZATION 기본값 설정

화면 뜨기 전 1회 — PARAMETERS/SELECT-OPTIONS 기본값.

Track 1Chapter 14Lesson 2중급

화면에 기본값을 채워두기

선택화면이 뜰 때 자주 쓰는 값(오늘 날짜, 기본 부서)이 미리 채워져 있으면 편하다. 화면 표시 전 1회 실행되는 에서 설정한다.

INITIALIZATION

PARAMETERS p_date TYPE d.
SELECT-OPTIONS s_dept FOR ztperson-dept_id.

INITIALIZATION.
  p_date = sy-datum.                       " 오늘 날짜 기본값

  s_dept-sign = 'I'. s_dept-option = 'EQ'.
  s_dept-low  = '1000'.
  APPEND s_dept TO s_dept.                  " 기본 범위 1건
  • INITIALIZATION은 화면 표시 전에 한 번 실행된다.
  • 여기서 PARAMETERS/SELECT-OPTIONS에 값을 넣으면 화면에 기본값으로 나타난다.

흔한 실수와 점검

  • START-OF-SELECTION에서 기본값 → 이미 화면을 지난 뒤라 화면엔 안 보인다. 기본값은 INITIALIZATION.
  • 여러 번 실행 기대 → 1회만. 매 화면마다는 AT SELECTION-SCREEN OUTPUT.

기본값은 채웠다. 그런데 화면을 그릴 때마다 필드를 숨기거나 잠그려면? → AT SELECTION-SCREEN OUTPUT(다음 레슨).
