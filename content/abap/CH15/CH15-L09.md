## 입력칸을 보기 좋게 배치

PARAMETERS·SELECT-OPTIONS를 그냥 나열하면 화면이 밋밋하다. **블록으로 묶고, 체크박스·
라디오·버튼**을 더해 쓰기 좋은 선택화면을 만든다. 모두 `SELECTION-SCREEN` 문으로 한다.

## 블록 · 구분선 · 설명

```abap
SELECTION-SCREEN BEGIN OF BLOCK b1 WITH FRAME TITLE TEXT-001.
  PARAMETERS p_carr TYPE sflight-carrid.
  SELECTION-SCREEN COMMENT /1(50) TEXT-002.   " 설명 줄
  SELECTION-SCREEN ULINE.                      " 구분선
  SELECTION-SCREEN SKIP.                       " 빈 줄
SELECTION-SCREEN END OF BLOCK b1.
```

- `BEGIN/END OF BLOCK ... WITH FRAME TITLE` — 테두리+제목으로 묶음.
- `COMMENT` 설명 텍스트 · `ULINE` 구분선 · `SKIP` 빈 줄.

여러 요소를 **한 줄에 나란히** 두려면 `BEGIN OF LINE`~`END OF LINE`으로 묶고 `POSITION`으로 칸 위치를 지정한다.

```abap
SELECTION-SCREEN BEGIN OF LINE.
  SELECTION-SCREEN COMMENT 1(10) TEXT-003.   " 1번 칸부터 라벨
  SELECTION-SCREEN POSITION 12.              " 12번 칸으로 이동
  PARAMETERS p_x TYPE i.
SELECTION-SCREEN END OF LINE.
```

- `POSITION n` — 다음 요소를 n번째 칸에 둔다. 범위 입력(SELECT-OPTIONS)의 From/To 자리는 `POS_LOW`·`POS_HIGH`로 가다듬는다.

## 체크박스 · 라디오버튼

```abap
PARAMETERS p_all AS CHECKBOX.                       " 체크박스

PARAMETERS: p_r1 RADIOBUTTON GROUP g1,              " 라디오 그룹 g1
            p_r2 RADIOBUTTON GROUP g1,
            p_r3 RADIOBUTTON GROUP g1.
```

- `AS CHECKBOX` — 켜고 끄는 한 칸(`'X'`/공백).
- `RADIOBUTTON GROUP g1` — 같은 그룹 중 **하나만** 선택.

## 본문 버튼 · 탭

- `SELECTION-SCREEN PUSHBUTTON /10(20) but1 USER-COMMAND cmd1.` — 화면 **안**의 버튼.
- `SELECTION-SCREEN BEGIN OF TABBED BLOCK ...` — 여러 화면을 **탭**으로 묶기.
- 각 탭의 내용은 **Subscreen**(화면의 일부 영역)으로 채운다 — 한 선택화면에 부분 화면을 끼워 넣는 구조다(탭마다 다른 입력 묶음).

## 애플리케이션 툴바 버튼 — FUNCTION KEY

화면 위쪽 **툴바**에 내 버튼을 달려면 `FUNCTION KEY`와 `SSCRFIELDS`를 쓴다.

```abap
TABLES sscrfields.
SELECTION-SCREEN FUNCTION KEY 1.       " 툴바 버튼 FC01

INITIALIZATION.
  sscrfields-functxt_01 = '새로고침'.   " 버튼 텍스트

AT SELECTION-SCREEN.
  CASE sscrfields-ucomm.
    WHEN 'FC01'.
      " 버튼 눌림 처리
  ENDCASE.
```

- `FUNCTION KEY 1~5` → `FC01~FC05`. `FUNCTXT_0n`으로 텍스트, `SSCRFIELDS-UCOMM`으로 눌림 감지.
- ⚠️ **본문 PUSHBUTTON ≠ 툴바 FUNCTION KEY** — 위치(화면 안 vs 위쪽 툴바)가 다르다.

## 흔한 실수와 점검

- **BLOCK 짝 안 맞음** → `BEGIN OF BLOCK`은 `END OF BLOCK`으로 닫는다.
- **라디오 그룹 미지정** → 같은 그룹이라야 배타 선택이 된다.
- **버튼 눌림 무시** → `SSCRFIELDS-UCOMM`을 `AT SELECTION-SCREEN`에서 분기.
