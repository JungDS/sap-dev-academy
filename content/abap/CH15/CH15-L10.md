## 옵션을 한자리에 모아 보기

여기까지 `OBLIGATORY`·`DEFAULT`·`AS CHECKBOX` 같은 옵션을 곳곳에서 만났다(CH03·CH12·
CH15-L09). 이 레슨은 **PARAMETERS와 SELECT-OPTIONS에 붙이는 옵션**을 한눈에 정리한다 —
필요할 때 찾아보는 **카탈로그**다.

## PARAMETERS 옵션

```abap
PARAMETERS p_amt TYPE p DECIMALS 2 OBLIGATORY DEFAULT '0'
                 LOWER CASE MEMORY ID zid MODIF ID grp.
```

| 옵션 | 뜻 |
|---|---|
| `TYPE` / `LIKE` | 타입을 직접 지정 / 기존 필드와 같게 |
| `DECIMALS n` | 소수 자릿수(타입 `p`) |
| `OBLIGATORY` | 필수 입력(비우면 진행 불가) |
| `DEFAULT <값>` | 초기값 |
| `LOWER CASE` | 소문자 보존(기본은 대문자 변환) |
| `AS CHECKBOX` / `RADIOBUTTON GROUP g` | 체크박스 / 라디오(CH15-L09) |
| `MEMORY ID <pid>` | **Parameter ID**와 연결 — 마지막 값 기억(CH03-L02) |
| `VALUE CHECK` | 입력값을 Foreign Key/고정값에 대해 검증 |
| `MODIF ID <grp>` | 화면 제어용 그룹 이름(아래) |

## SELECT-OPTIONS 옵션

```abap
SELECT-OPTIONS s_date FOR sflight-fldate
  DEFAULT '20240101' TO '20241231'
  OBLIGATORY NO-EXTENSION NO INTERVALS
  MEMORY ID zdt MODIF ID grp.
```

| 옵션 | 뜻 |
|---|---|
| `DEFAULT <low> [TO <high>]` | 초기 단일값 / 초기 범위 |
| `OBLIGATORY` | 최소 한 줄 필수 |
| `LOWER CASE` | 소문자 보존 |
| `NO-EXTENSION` | **다중 선택(여러 줄) 버튼 숨김** — 한 줄만 |
| `NO INTERVALS` | From–To 중 **To 칸 숨김**(단일값만) |
| `MEMORY ID` / `MODIF ID` | 값 기억 / 화면 제어 그룹 |
| (SIGN·OPTION) | 부호·비교연산자 — Range Table(CH12) |

## MODIF ID — 한 그룹을 한꺼번에 제어

같은 `MODIF ID grp`를 준 항목들을, 화면이 뜨기 직전 **묶어서** 보이기/숨기기·잠그기
할 수 있다.

```abap
AT SELECTION-SCREEN OUTPUT.            " 화면 뜨기 직전
  LOOP AT SCREEN.
    IF screen-group1 = 'GRP'.          " MODIF ID가 GRP인 항목들
      screen-active = '0'.             " 0=숨김 (1=표시)
      MODIFY SCREEN.
    ENDIF.
  ENDLOOP.
```

- `AT SELECTION-SCREEN OUTPUT` — 선택화면이 그려지기 직전 이벤트(화면을 동적으로 손본다).
- `MODIF ID`로 묶고 `LOOP AT SCREEN`에서 `screen-active`/`screen-input`을 바꿔 한 번에 제어.

## 흔한 실수와 점검

- **MEMORY ID 충돌** → 같은 Parameter ID를 엉뚱한 필드에 쓰면 값이 섞인다. 고유하게.
- **NO INTERVALS인데 범위 입력 기대** → To 칸이 없다. 범위가 필요하면 빼야.
- **MODIF ID 제어가 안 먹음** → `AT SELECTION-SCREEN OUTPUT`에서 `MODIFY SCREEN`까지 해야 반영.
