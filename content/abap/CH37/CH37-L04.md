## 화면 출력을 넘어 — 파일로

Adobe Forms는 PDF를 **바이트(XSTRING)** 로 돌려준다(L02의 `formoutput`). 이걸 저장·이메일
첨부·DMS 보관에 쓴다.

```abap
DATA(lv_pdf) = ls_formout-pdf.        " XSTRING(PDF 바이트)

" XSTRING → 바이너리 테이블 후 PC 저장
DATA lt_bin TYPE solix_tab.
lt_bin = cl_bcs_convert=>xstring_to_solix( lv_pdf ).
cl_gui_frontend_services=>gui_download(
  EXPORTING filename = 'ticket.pdf' filetype = 'BIN'
            bin_filesize = xstrlen( lv_pdf )
  CHANGING  data_tab = lt_bin ).
```

- PDF는 텍스트가 아니라 **바이너리** — `BIN` 모드로 저장.
- 이메일 첨부는 BCS(`CL_BCS`)로, 보관은 ArchiveLink/DMS로.

## 흔한 실수와 점검

- **TEXT 모드 저장** → PDF가 깨진다. 반드시 `BIN`.
- **인코딩 변환 실수** → XSTRING↔바이너리 변환 헬퍼를 쓴다.
