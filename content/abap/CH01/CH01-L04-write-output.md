## WRITE로 화면에 출력하기

_작성 예정: [[WRITE]] 한 줄로 문자열을 출력하고 F8로 결과 화면 확인._

```abap
REPORT zhello.

WRITE 'Hello, ABAP!'.
```

## 여러 줄 출력과 줄바꿈

_작성 예정: `/` 로 줄을 바꿔 출력._

```abap
WRITE 'first line'.
WRITE / 'second line'.
```

## 문자열 리터럴

_작성 예정: 작은따옴표 `'...'` 문자 리터럴, 아직 변수는 없다(→ CH02 불편의 씨앗)._

## 흔한 실수와 점검

_작성 예정: 따옴표 짝 안 맞음, 마침표 누락._
