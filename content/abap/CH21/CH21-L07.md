## 갱신과 사용자 경험

예매가 바뀔 때마다 그리드를 새로 그리되, 사용자가 **보던 위치·선택·정렬**이 초기화되면
불편하다. CH17의 Stable Refresh를 심화한다.

```abap
DATA ls_stable TYPE lvc_s_stbl.
ls_stable-row = abap_true.        " 행 위치 보존
ls_stable-col = abap_true.        " 열 위치 보존

go_grid->refresh_table_display(
  EXPORTING is_stable = ls_stable
            i_soft_refresh = abap_true ).   " 정렬·필터까지 보존(soft)
```

- `is_stable` — 스크롤·선택 보존.
- `i_soft_refresh` — 정렬·필터 등 표시 상태까지 가볍게 유지.

## 흔한 실수와 점검

- **매 갱신 hard refresh** → 정렬·위치가 다 풀린다. 편집·갱신 잦으면 stable/soft.
- **데이터 동기화** → 화면 테이블과 내부 테이블이 같은 데이터를 가리켜야.
