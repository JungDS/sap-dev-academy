## 표시 설정 다듬기

SALV 객체에서 제목·줄무늬·컬럼 속성·레이아웃 저장을 코드로 조정한다.

```abap
lo->get_display_settings( )->set_striped_pattern( abap_true ).   " 줄무늬
lo->get_display_settings( )->set_list_header( '회차별 예매 현황' ).

DATA(lo_cols) = lo->get_columns( ).
lo_cols->set_optimize( abap_true ).                              " 너비 최적화
lo_cols->get_column( 'CAPACITY' )->set_long_text( '정원' ).      " 컬럼 제목
```

## Variant 저장

```abap
lo->get_layout( )->set_key( VALUE #( report = sy-repid ) ).
lo->get_layout( )->set_save_restriction( ).                       " 사용자 레이아웃 저장 허용
```

## 흔한 실수와 점검

- **컬럼명 오타** → `get_column( )`은 존재하는 컬럼명이라야.
- **레이아웃 키 누락** → Variant 저장하려면 `set_key`로 식별.
