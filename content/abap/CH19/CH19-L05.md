## DB에서 문자열·날짜도 가공

modern SQL은 문자열·날짜용 함수도 제공한다. 자주 쓰는 것만 익혀 둔다.

## 문자열 함수

```abap
SELECT carrid,
       CONCAT( carrid, connid )        AS route,     " 잇기
       SUBSTRING( connid, 1, 2 )       AS prefix,    " 잘라내기
       UPPER( cityfrom )               AS city_up    " 대문자
  FROM spfli
  INTO TABLE @DATA(lt).
```

## 날짜 함수

```abap
SELECT carrid, connid, fldate,
       DATS_ADD_DAYS( fldate, 7, 'INITIAL' ) AS due  " 7일 뒤
  FROM sflight
  INTO TABLE @DATA(lt_flight).
```

- `CONCAT`·`SUBSTRING`·`UPPER`/`LOWER`·`LENGTH` 등 문자열 함수.
- `DATS_ADD_DAYS`·`DATS_DAYS_BETWEEN` 등 날짜 함수.
- 함수마다 DB·릴리스에 따라 지원이 다를 수 있어, **필요할 때 찾아 쓰는** 식으로.

## 흔한 실수와 점검

- **모든 가공을 SQL로** → 복잡하면 ABAP/CDS가 낫다. SQL 함수는 단순 가공.
- **별칭 권장** → 함수 결과 컬럼엔 `AS`로 이름을.
