## @ вҖ” "мқҙкұҙ ABAP к°’мқҙлӢӨ"

modern SQLм—җм„ң ABAP ліҖмҲҳВ·мғҒмҲҳВ·н•„л“ңлҘј SQLм—җ л„Јмқ„ л•җ м•һм—җ **`@`**лҘј л¶ҷмқёлӢӨ. мқҙлҘј
**host variable**мқҙлқј н•ңлӢӨ.

```abap
SELECT * FROM sflight
  WHERE carrid = @lv_carr
    AND fldate = @lv_date
  INTO TABLE @lt_flight.
```

## @( ) вҖ” мӢқлҸ„ нҶөм§ёлЎң

лӢЁмҲң ліҖмҲҳлҝҗ м•„лӢҲлқј **мӢқ**(кі„мӮ°В·л©”м„ңл“ң кІ°кіј)лҸ„ `@( ... )`лЎң л„Јмқ„ мҲҳ мһҲлӢӨ(host expression).

```abap
SELECT * FROM sflight
  WHERE seatsmax > @( lv_base + 100 )
  INTO TABLE @lt_flight.
```

- `@lv_x` вҖ” ABAP ліҖмҲҳ н•ҳлӮҳ.
- `@( мӢқ )` вҖ” кі„мӮ°мӢқВ·мғҒмҲҳ н‘ңнҳ„ л“ұ.

## нқ”н•ң мӢӨмҲҳмҷҖ м җкІҖ

- **м»¬лҹјм—җ @** вҶ’ DB м»¬лҹј(`carrid`)м—” `@`лҘј м•Ҳ л¶ҷмқёлӢӨ. ABAP к°’м—җл§Ң.
- **@( ) кҙ„нҳё лҲ„лқҪ** вҶ’ мӢқмқҖ л°ҳл“ңмӢң `@( ... )`лЎң к°җмӢјлӢӨ.

---

ABAP к°’мқ„ м•Ҳм „н•ҳкІҢ л„Јм—ҲлӢӨ. мқҙм ң **кІ°кіј л°ӣмқ„ н…Ңмқҙлё”лҸ„ мқёлқјмқёмңјлЎң** м„ м–ён•ҳмһҗ вҶ’ лӢӨмқҢ л ҲмҠЁ.
