## CONCATENATE를 넘어서

CH04에서 `CONCATENATE`·`&&`로 문자열을 이었다. 모던 **String Template**(`| … |`)은
중괄호 `{ }`로 변수를 끼워 훨씬 읽기 좋게 만든다.

```abap
" classic
CONCATENATE lv_name '님 환영합니다' INTO lv_msg.

" modern
DATA(lv_msg) = |{ lv_name }님 환영합니다|.
```

## 서식까지 한 번에

`{ 값 옵션 }`으로 날짜·숫자 서식을 바로 준다.

```abap
DATA(lv_line) = |오늘은 { sy-datum DATE = USER }, 합계 { lv_total NUMBER = USER }|.
DATA(lv_eq)   = |{ ls-dan } x { ls-mul } = { ls-result }|.
```

## 함수형 문자열 함수

CH04의 명령형 문자열 처리를, 모던에서는 **값을 돌려주는 함수**로 쓴다.

```abap
DATA(lv_up)  = to_upper( lv_name ).
DATA(lv_sub) = substring( val = lv_text off = 0 len = 3 ).
DATA(lv_len) = strlen( lv_text ).
```

## 흔한 실수와 점검

- **`|` 안에서 `{ }` 누락** → 변수는 `{ }`로 감싸야 값이 들어간다.
- **숫자 직접 임베딩 공백** → 숫자는 서식 옵션(`NUMBER = USER`)으로 다듬는다.
