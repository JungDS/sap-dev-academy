## 데이터(속성)와 행동(메서드)

클래스는 **속성**([[Attribute]], 데이터)과 **메서드**([[Method]], 행동)를 가진다.

```abap
CLASS zcl_booking_manager DEFINITION PUBLIC.
  PUBLIC SECTION.
    METHODS book IMPORTING iv_seats TYPE i.
    METHODS remaining RETURNING VALUE(rv) TYPE i.
  PRIVATE SECTION.
    DATA mv_capacity TYPE i.          " 인스턴스 속성
    DATA mv_booked   TYPE i.
ENDCLASS.
```

- `DATA` — **인스턴스** 속성(객체마다 따로). `CLASS-DATA` — **정적** 속성(클래스 공용).
- `mv_` 접두어는 멤버 변수 관례.

## 공개 범위 — Visibility

| 섹션 | 누가 접근 |
|---|---|
| `PUBLIC SECTION` | 누구나(외부 포함) |
| `PROTECTED SECTION` | 자신 + **상속받은 자식** |
| `PRIVATE SECTION` | **자신만** |

- 외부에 보일 것(메서드)은 PUBLIC, 내부 데이터(잔여석 계산용)는 PRIVATE에 **숨긴다**(캡슐화).

## 흔한 실수와 점검

- **데이터를 PUBLIC** → 캡슐화가 깨진다. 데이터는 보통 PRIVATE, 접근은 메서드로.
- **인스턴스 vs 정적** → 객체마다 다른 값=`DATA`, 클래스 공용=`CLASS-DATA`.
