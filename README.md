# SAP Developer Academy

> 개발자 시선으로 **0부터 안내하는** SAP 학습 여정.

SAP 개발에 입문하는 사람을 위한 guided·**동기부여형(motivation-first)** 학습 사이트입니다.
"객체 분류 순서로 쌓는" 기존 방식 대신, **불편을 먼저 체감하고 → 그 해결책으로 개념을 도입**하는
순서로 ABAP을 비롯한 SAP 개발 역량을 단계적으로 익힙니다.

## 학습 도메인 (멀티 도메인 허브)
- **ABAP 커리큘럼** — Chapter 1(SAPGUI/SE38 첫 실행)부터 시작하는 동기부여형 본 과정
- **UI5 / Fiori** — 화면/UI 개발 (예정)
- **SAP 모듈** — FI/CO/MM/PP/SD를 *개발자 관점*에서 (예정)
- **통합 실습** — ABAP↔Gateway↔UI5 Full-Stack 시나리오 (예정)

글로서리(용어집)·T-code 지도 등 참고 도구는 독립 도메인이 아니라 커리큘럼 전반을
가로지르는 **보조 레이어**로 제공됩니다.

## 구조
```
index.html        멀티 도메인 허브 랜딩
assets/           CSS/JS (점진적으로 구축)
docs/             레슨·가이드 콘텐츠 (점진적으로 구축)
pages/            도메인별 허브 페이지 (점진적으로 구축)
```

## 상태
초기 부트스트랩 단계입니다. 콘텐츠는 깨끗한 백지에서 **점진적으로** 재구축합니다.

## 비고
이 프로젝트는 구 저장소 `JungDS/sapui5`("SAP Developer Learning Library")를 대체합니다.
구 저장소는 역사적 기록·참고 원본으로 그대로 보존되며, 필요한 자산은 점진적으로 이식합니다.
